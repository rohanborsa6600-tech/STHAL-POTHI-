संसरण

लापिका :-अनादि मायास्वरुपीं तनु-मच्छवत् तम असे ।१। ऐसा प्रपंचाचा संहार जालेयानंतरें जीव कवणें ठाइं असति पां? ना अनादि अविद्या कर्मलेपयुक्त जीव मायास्वरुपीं असति : तरि मायास्वरुप सृष्टीमध्यें व्यापलें असे तेथचि असति आणि काइ? ना मायास्वरुपीं एकदेशी तम असे ।२।
वचन :- मायास्वरुपीं एकदेशी तम असे ॥
टीका :- माया कां : मोहिलें : भ्रमविलें : अन्यथाज्ञान उपजविलें : पोटीं वाहिलें : जनिलें : संख्ये विषयो नव्हे तो एकदेश : यथा पृथ्वी परमाणुवत् : डोळेयांत काळिमा तैसें तम एके सेवटीं : तम, अवगळा, नित्य, फळ ब्रह्मांडावेगळें : कष्मळ तें एके कोनीं असे : दृष्टी पुढां नसे : जैसा पुंजा एके कोनीं घालिजे : दृष्टी पुढां चोखट : जैसें विरक्ताचिया दृष्टीपुढां पदार्थ पडिला परि से नाहीं : तैसें अव्यक्तकार्य प्रवर्ते : जैसा माया तमीं भ्रमविला असे : पाळणा घालौनि हालवी मग निद्रा ये :॥:॥: तम म्ह. काळिमा : यथा लाहासें : बुबुळ : स्वरुपीं ज्ञान सुख सामथ्र्य असौनि विवर्णता : ॥१॥

लापिका :- तिये तमीं मज्जित अनंत जीव असति : ॥
वचन :- तमोनिमग्न जीव मायास्वरुपीं असति ॥
टीका :- निमग्न म्ह. हारतले : यथा समुद्रीं जळचरें : ऐसें अनंत जीव हारतलें जिये तमीं : ते तम जियेचां एकदेशीं : मग ते माया जयाचां एकदेशीं : तयापरौता जो परमेश्वर आणि तेसणाचि जीव : हे थोरी जीवें जाणावी :॥:॥: असति म्ह. यथावकाशें : आतां घातले ; सवेंचि काढिजति हे नाहीं :I।: संलग्नासि फळीं, संहारी सारिखाचि भोग कीं : यथा स्तनीं काळें ठीक तरि काइ दुधाची गोडी न फवे : तैसें तया तमांत सुख :।I:०:I।: महासंहारी अवघें एकवट (वाट) कां नेइजति (सुइजति) : जेथचे तेथचि कां नसावे पां : मग तेथौनिचि मळ घेति : जें देवता स्वरुपस्थिता म्हणितलेयां म्हणौनि : तरि जेथचे तेथ असिलेयां एणें फळत्व न मोडे म्हणौनि :।I: ॥२॥

लापिका :- परमेश्वरकृपा प्रवृत्तीमात्रें चिन्माया तमस्थिता जीवां तथा संलग्ना चैतन्यमहमिति उपजवी : ॥
वचन :- परमेश्वरप्रवृत्ती माया तमोनिमग्ना जीवा चैतन्यमहं ऐसे उपजवी ॥
टीका :- परमेश्वर म्ह. समर्थ : भलतें (भलेंतें) करू शके : जीवाचा ईश्वरही :।: प्रवृत्ती म्ह. उल्लेख :।: मोटका मोहरैजे : तमोनिमग्न म्ह. जे तमीं निमज्जन पातले असति : जीवा म्हणितलेया चैतन्यही लाभे : दोहीं सारिखें उपजवी : एकें काळीचि मळ लागे : म्हणौनि दोहीं एक वाक्य : चैतन्यमहं म्ह. चैतन्य तें मी : जीव संलग्नातें म्हणें : तें मुख्यातें म्हणें : उपजवी म्ह. अन्यथाज्ञानरुप उल्लेख :॥: चैतन्य म्ह. आपण चेतन : जीवातें चेतवी म्ह. सावधा करी :॥: सृष्टीकाळीं जें चैतन्य ज्ञानरुप होतें तें जीवासरीसे अज्ञान केलें मग दोहींसि अन्यथाज्ञान उपजविलें :I।: संलग्ना आपुलेया स्वरुपातें मी एवढें ऐसें अन्यथाज्ञान उपजवी :I।: अव्यक्तें ईश्वरी उल्लेख उठी : पाते पिसी (पुसी) : यथा ईश्वरीची प्रवृत्ती आणि तियेचा व्यापार : अहं हे काइ ईश्वरांतवं नुपजववे : यथा दासी सेण फेडी :।I: जीवोहं कां नुपजवी : यथार्थ ज्ञानें न मैळे : विश्वमहं कां : वरील मळ ठाकैल : ईश्वरोहं कां : तें उपजउ न ये : दासधर्म नव्हे :।I: ॥३॥

लापिका :- तरि काइ उपजवी एतुलेंचि? ना मानसिक शारीर व्यापारें सकळ निष्पत्ती करुनि लोटी म्ह. चतुर्विध फळपर्यंत : ॥
वचन :- माया जीवांते विकरे : संचरे : विये : प्रसवे : हालवी : डोचली : उगळी : लोटी ॥
टीका :- माया आपण विकरें : जीवातें विकरवी म्ह. (म्हणौनि) अनारिसेयां करी :॥: माया जीवातें विकरें हा प्रथम उल्लेख : मग हाही विकरे विकारीं रस जैसा :॥: माया विकरे हा मनोधर्म : एरें सात वचनें शारीर व्यापार : कां जें पुढा उदासीन : शारीरचि निराकरिला : आणि सूत्र हा मनोधर्म :॥: संचरे म्ह. जिये स्थानीं असति तेथौनि परतें सारी : जैसा चौधार गुंडा जिये वाहानी केला :॥: विये म्ह. योनिपासि आणी : खोळि सांडी : जैसी गाय विये :॥: प्रसवे म्ह. आधे आंत : आधे बाहीरि : जैसें सीस खुर बाहीरि पडति वांसरुवाचें :॥: हालवी म्ह. सरिसें करी : निवडी जिये ब्रह्मांडीचें तिये ब्रह्मांडी : एकवट होते की : जैसें बाण भाताचे सरिसें किजति :॥: डोचली म्ह. सावधां करी : मागिला अन्यथाज्ञानाचा अभिन्नवेष :॥: उगळी म्ह. तमाबाहीरें घाली : योनि बाहीरें घाली : जैसें सर्प आविष बाहीरें घाली :॥: लोटी म्ह. जिये ब्रह्मांडीचे तेथ लोटी : जड जेथचे तेथ : स्वरुपाचेनि आधारें :॥: लोटी म्ह. यथा स्थानीं घाली : जो जेथ चेतन, जड : ऐसा आकार : यथा सुतार फळी रेखी मग घडी : ॥४॥

लापिका :- ऐसें उपजविलियांउपरी जीव काइ करीति पां? ना माया व्यापाराधीन होउनि अविद्यात्मकत्वें चैतन्य तें मीं ऐसें म्हणति : तरि हें वचन तेथ होआवें की? ना माया मानसिक व्यापाराचिये पांति तियेचा सकळ शारीर सांगितला : मग जीवाचा : म्हणौनि हें वचन एथ : ॥
वचन :- परमेश्वरप्रवृत्ती माया-व्यापारवशें अविद्यात्मक जीव चैतन्यमहं ऐसे घेउनि उठीति ॥
टीका :- माया-व्यापारवशें म्ह. व्यापारासि अधीन होउनि : अविद्यात्मक म्ह. अविद्याचि आत्मा केला असे : चैतन्यमहं म्ह. चैतन्य तें मीं : घेउनि उठीति म्ह. अहंभावेंसीं सावध होति : यथा निद्रीस्थ मागिलें अध्यासेंसींचि उठीं :॥: मीं चैतन्य या प्रथम उल्लेखाचा मळ जनला : यथा प्रेतदेह : परि विशेषज्ञानेंहीं न नसे : तो ईश्वरें नासावा प्राप्तीकाळीं का संहारीं :I।: प्रतिसृष्टी चैतन्य जीवासि आन आन लागे की एकचि : आन आन नव्हे : केवि : जे चतुर्विध कर्मफळें तरि स्वर्ग जयासि होआवें तयासि नरक होति : नरक जयासि होआवें तयासि स्वर्ग होति : म्हणौनि पालट नव्हे : एकीचि फळीं तरि विचित्रता असे म्हणौनि पालट नव्हे : यद्यपि विचित्रता नाहीं पां तरि पालट नव्हे : फळच्युति जालियां पुढति आन आन भोगी : म्हणौनि जीवा चैतन्या पालट नाहीं : जीवाचां मोक्षीं चैतन्या मोक्ष तो पालटचि कीं :।I: ॥५॥

लापिका :- तरि चैतन्यमहं म्हणितलेया काइ जालें पां? ना एणें अन्यथात्वें जन्माद्यमळ लागला : परि तमांतचि लागला ऐसे नाहीं : ॥
वचन :- चैतन्यमहं जीवा जन्म बोलिजे : अन्यथाज्ञान बोलिजे : आद्यमळ बोलिजे ॥
टीका :- जीवा जन्म बोलिजे म्ह. मी चैतन्य म्हणितलें तेणेंचि जन्म पातला : तेणेंचि मळ लागला : आदि म्ह. अवघेंया मळा आदि लागला म्हणौनि : अवघेंया संसरणा आदि म्हणौनि आदिमळ : अवघेंयाची आदि तेणें लाविला म्हणौनि आदिमळ : मळ म्ह. संकोची, मैळी : आद्यमळ लागलेया जीवासि आल्हाद आति : खेद नाहीं : मनुष्यमळपर्यंत : स्थुळें प्रपंची खेदही आति : आणि अहं यातेचि अन्यथाज्ञान म्हणिजे : बोलिजे म्हणितलेया तीहीतें एकें शब्दें : ॥६॥

लापिका :- एवं सृजनाद्य आद्यमळ माया जीवासि लाविला मग जीवें काइ केले पां? ना विश्वमळादि सकळ प्रपंच रचीला : ॥
वचन :- मायासृष्टी जीव : जीवसृष्टी प्रपंचु ॥
टीका :- मायासृष्टी म्ह. सृजनांत सृष्टी : एक आथिले सृजिलें तें जीव : एक नाथिलें/ला सृजिलें/ला तो प्रपंच : जैसे हें तुरक एथ कै सृजिलें : जीव तमींचें ब्रह्मांडी सृजिलें : प्रपंच नाथिला जीवें सृजिला :॥: दुसरा अर्थ :।: सृष्टी म्ह. सृजनक्रिये विषयो जाला जो पदार्थ आद्यमळ : जीवसृष्टी म्ह. तथा सृजनक्रिये विषयो जाला जो विश्वमळ : परंपरा स्थुळपर्यंत : ॥७॥

लापिका :- तरि काइ आद्यमळ लाविला एतुलेंचि? ना सामथ्र्य देउनि समर्थ केला : मग जीवें अविद्यमान प्रपंच केला : ॥
वचन :- मायाकृत जीव : जीवकृत प्रपंचु ॥
टीका :- मायाकृत म्ह. केले : निमित्तत्वें : यथा कुलाळवत् : जीवत्व काइसेयाचें केलें : तरि समवाय निराकरिला : कार्य म्हणितले यातवं :।: तथा जीवकृत प्रपंच : एवं समवाय उपादान ये दोन्हीं कारणें निराकरिली : नसुद्धे निमित्तचि :॥: दुसरा अर्थ :।: मायाकृत म्ह. जीव मेले ऐसें होते ते जीवविले : रुपासि आणिलें : जैसें हें मनुष्य आमुकेनि केलें : जीवकृत प्रपंच म्ह. पंचभौतिक त्रिगुणात्मक प्रपंच हाही मागिलाचि परी जाणावा :॥: तिसरें : मायाकृत म्ह. मतीजड ऐसा होता : अविद्यमान प्रपंच केला : तैसा तोही केला : स्थुळा प्रपंचेसी जीवाचिये दृष्टी गोचर नव्हे तो तया मेला : सूक्ष्मेसी देवताचिये दृष्टी विखो होत असे : अविद्येसी मायेचिये दृष्टी : केवळ तैं देवतांसीं मेला : ईश्वराचिये दृष्टीसि तैं मायेसि मेला : संहारी देवतांसि मेला म्हणावा : एऱ्हवीं जीव जीतुचि असे : सूक्ष्मदेह कहींचि न संडी : ॥८॥

लापिका :- ऐसें उभय कतृ©त्व सांगितले तरि हें पदार्थ अवांतर ऐसें लाभैल तें कार्य शब्दें निराकरिलें : एवं अनादिचे स्वतःसिद्ध : ॥
वचन :- मायाकार्य जीव : जीवकार्य प्रपंचु ॥
टीका :- मायाकार्य म्ह. जयापासौनि जें उपजे तें तयाचें कार्य म्हणिजे : जीव कार्य माया कारण : तैसेंचि जीव कारण प्रपंच कार्य :॥: प्रपंच कार्य जीव कारण : जीव कार्य माया कारण : माया कार्य ईश्वर कारण : यदनतरं यद्दृश्यते : ॥९॥

लापिका :- एवं माया जीवातें सिद्ध करुनि काइ करी पां? ना उदास होए : परि मनोधर्म असतुचि असे : ॥
वचन :- या परी माया जीवांते जन्मौनि आपण उदासीन होए : परि सुत्राते न संडी ॥
टीका :- या परी म्ह. शारीर मानसिक व्यापारें : जन्मौनि म्ह. सामथ्र्य देउनि तमाबाहीरि घातलें : उदासीन म्ह. शारीर व्यापार राहिला : सुत्रातें न संडी म्ह. मनोधर्म असे : यथा पुत्र वृद्ध जालेयां मातेचें आधील करण राहे : मग तो स्वातंत्रियें उठे बैसे : परि मनोधर्म असे : झणें अवजतन होए :॥: दुसरें सूत्र म्ह. यथा प्रधान सकळ म्हणियाचां ठाइं : उचा निचाचां ठाइं : कां हें पतीचें तेणें भरलेपणे पाहे : ॥१०॥

लापिका :- तरि जन्मलेयाउपरि जीव कवणें ठाइं असे पां? ना माया स्वरुपाधारें अथवा प्रपंच निर्वाळुनि स्थुळी असे : मध्यें असणें (असणेंचि) नाहीं : ॥
वचन :- जीवु मायेसि अधीनु होए : का प्रपंचासि अधीनु होए : मध्ये असणेंचि नाहीं ॥
टीका :- मायेसि अधीन म्ह. जैसें उदक नैसि : विहिरीसि : अडासि :॥: अधीन म्ह. दों आधार : मध्यें म्ह. दों विरहित : स्वातंत्रियें सूक्ष्मेंसीं असणें नाहीं : आधारभूतेविन : ॥११॥

लापिका :- तरि काइ जन्मत क्षणीचि प्रपंचातें रची? ना आद्यमळेंसी क्षण एक (शुद्ध) असे मग प्रपंच (रचना) वासना मैळे : ॥
वचन :- जन्मला सांता जीवु क्षणपर्यंत शुद्ध असे मग मैळे ॥
टीका :- जन्मला म्ह. अविद्येसि मायेसि : सूक्ष्मेंसी देवतांसि : स्थुळेसी जीवांसि :॥: दुसरें : जन्म म्ह. तमाबाहीरि लोटीलें हें एक जन्म : तेणें जन्मलेपणें : क्षण म्ह. घडिये आंतुल विभाग : सा पानीवळीं : एका पानीवळिया स्वासोस्वास सा : एवं छत्तीस : शुद्ध म्ह. प्रपंचापेक्षा : तें आनचि जाति : यथा कापुर परीमळाचा बांधा आणि निरवयव :I।: शुद्ध पहिलें : वासना मैळे : मग कार्य विश्वमळ रची : तवं निका न मैळे : मग चांग : रोटियेची वासना मग खाये :।I: ॥१२॥

लापिका :- तरि काइ (प्रपंच) मळींचि मैळे? ना असत्कर्मेंहीं मृत्तिकापर्यंत मैळे : परि संलग्नाचेनि आधारें : ॥
वचन :- चैतन्य मृत्तिकापर्यंत मैळे : चैतन्यासवे जीवु मृत्तिकापर्यंत मैळे ॥
टीका :- चैतन्य म्ह. संलग्न : मृत्तिका म्ह. स्थुळ परमाणु : पर्यंत म्ह. सेवट मळिनत्वाचा : हे सीमा कर्मराहाटीं :॥: चैतन्यासरिसा जीव म्ह. सकळ क्रिया तें पुढां : कां जें जीव पंगु, आंधळा : तेणेंविन यासि गतीचि नाहीं : यथा डोळसाचा हात धरलेयाविन अंधासि गतीचि नाहीं : संसरणीं जीव पुढां : तें मागां : कां जें तैं सामथ्र्य होतें : तें गेलेयांवरि विपर्याय म्हणौनि चैतन्यासरिसा म्हणितलें :॥: मैळे उजळे या दोहीं समानत्व आपजैल म्हणौनि : अधमति ते सर्वदा : तरि मैळे हें बाहुल्यें : उजळे हें सकृत : कां जेतुलीं पापफळें तेतुलीं पुण्यफळें नाहीं कीं :I।: तराळें आपणेया हातखोडा घातला : तैसा चैतन्यें आपणेया बंध लाविला :।I: ॥१३॥

लापिका :- तथा सत्कर्में चैतन्यपर्यंत उजळें परि तदाधारेंचि : ॥
वचन :- चैतन्य चैतन्यपर्यंत उजळे : चैतन्यासवे जीवु चैतन्यपर्यंत उजळे ॥
टीका :- चैतन्य म्ह. संलग्न : चैतन्यपर्यंत म्ह. चैतन्यफळाची सीमा : कर्मभूमि लागौनि तो ठावोवेऱ्हीं चैतन्या उजळणें आति : उजळें म्ह. चोखट होए : विद्या क्षाळणेंकरौनि : असंतें मैळे : संते उजळें : मां तें देवता कीं : मैळणें तें भोगें की : तया काइ भोग असे : ना क्रिया दोहीं सारिखी भोगितां परि दुःख नाहीं : तें जीवासिचि : यथा लोहेंसीं अग्नी तैसा ठावो : तयासि मळीं मळीनत्व : कर्मी नाहीं : कां तया लेप न लगे : आणि जीव दुःखीं मैळे : सुखीं उजळे : अंतर्यागीं बहिर्यागीं :॥: तयासरिसा म्ह. तेयाचेनि आधारें : कां जें जीव सर्वदा परतंत्र : बोधकाळीं अनुसरला स्वतंत्र : वासनिक उचिते व्यापारीं स्वतंत्र : कर्मव्यापारीं परतंत्र : चैतन्यादि भावो सर्वदा : वासनिकांही आदिकरौनि उचिते अनुचिते अर्थीं, बोधां अनुसरणीं चैतन्या पश्चात भावो :I।: दुसरें : चैतन्य मृत्तिका ज्ञानें उजळें मैळे : मग फळीहीं : आत्मज्ञानें मोक्ष :।I: ॥१४॥

लापिका :- हो का : संलग्नाधारें म्हणितले तरि अधिकार्या कवण पां? ना तो जीवचि : एरा तदाभिमान तो वृथा : तरि यें वचनें पुढां असावी कीं? ना प्रपंचकार्याचिये पांति कर्मकार्य सांगितले : परि विनियोग तो पुढां : ॥
वचन :- आनाचां करणां तयासि अभिमानु ॥
टीका :- आन म्ह. जीव : वायपटचि अभिमान : चैतन्य तें लेपाधिकार्ये नव्हौनि परि सुखदुःखें सरिसे भोगी : जीवाचेनि भोगें तयासि भोग : अधिकार्या जीवचि : जैसा कव्हणीं एक वाराणसी घोडेया वोळघौनि गेला : ते क्रिया दोघीं केली परि पुण्य तें कोणा :॥: दुसरें : आनाचा म्ह. स्वरुप वैजात्य : लेप समग्र भोगी : तो नेणो कोण लोक : सेजा गोत्रियु नव्हें : जाति नव्हे : चैतन्या जीवाचा अभिमान : करण जीवाचे : ॥ शोध ॥ आनाचा करणा तया म्ह. जीवा : एकी वासना : तया म्ह. चैतन्या : ॥१५॥

लापिका :- तरि जन्मानंतरे क्षणपर्यंत शुद्ध असे मग काइ करी पां? ना वासना मैळुनि प्रपंचातें रची : ॥
वचन :- मग जीवु प्रपंचाते रची ॥
टीका :- प्रपंचाते रचीति म्ह. मळ घेति : काइ एकें वेळें : ना अनुक्रमें :I।: चैतन्यफळ पातलेयां संहार जाला : सृष्टीकाळीं मळ जीव एकलाचि करी : ऐसें सामथ्र्य असे प्रपंचकरणी : मायेसि जीवकरणी : म्हणौनि चैतन्याचे बळ न लगे :।I: जो मळ करी तयाचि पुरतें सामथ्र्य असे : तादात्म्य पावें आणि जाए : जैसी कोसकीडी :I।: देवतेचा मळ ऐसें कां म्हणितलें : ना तेणें मळें वांचौनि देखेना म्हणौनि : मळ देवते न लगे : कां न लगे : देवते प्रपंच होववे ना : कां जीवकृत म्हणौनि : चैतन्यासि निफजे : केवि : नासित असे म्हणौनि रचू सके : एरी न सकति : स्थुळ आठै संहारीति म्हणौनि रचू सकति : एरी स्थुळहीं नासू न सकति : तयाचे सामथ्र्य सांगितले म्हणौनि :I।: लेप चैतन्याचेनि लागेना पुसेना : कव्हणाचेनिही : ईश्वराचेनि लागत असे : पुसत असे : कृमिचेनि तें कृत्य म्हणौनि :।I: ॥१६॥

लापिका :- तरि समळा प्रपंचातें कां पां रची? ना मतित्रयसंभवे हीनचि आवडें : ॥
वचन :- कां रची जे अधमति : अधगति : अधरति म्हणौनि रची ॥
टीका :- प्रतिमळीं : पाठीं कां रचीति : कैसेनि रचीति : एक मळ करी मग दुसरा करावेया बुद्धी, सामथ्र्य नाहीं : तोचि दुसरा नव्हवें :॥: रचीति म्ह. मति : रिगे म्ह. गति : तादात्म म्ह. रति : बुझला म्ह. मळाचें आत्मत्वें दर्शन : तें तवंचि जवं आणिकें मळीं तादात्म होए :॥: अधमति करुनि घेतला : उर्धगति तो कर्मवशेंचि होआवा : जैसा वोहटळेचा गुंडा : यथा गंगोदक : ॥१७॥

लापिका :- तरि असमर्थ जीव काइसेनि रची पां? ना माया सामथ्र्य दिधलें तेणें : हा शंका परिहार : ॥
वचन :- काइसेनि रची : तै तया सामथ्र्य आति ॥
टीका :- काइसेनि रचीति म्ह. मागिलासि आशंका : कां जें जीव शोच्य तरि रची तें काइसेनि : तो परिहार केला पुढिलें शब्दें :॥: तैं म्ह. सृष्टीकाळीं : तया म्ह. जीवासि : सामथ्र्य म्ह. करिजे ऐसी शक्ती आति :।I: मळ क्षाळणीं प्रतिमळीं ईश्वर हेतू : आणि संसरणीं नाहीं : कां जें तैं तैसें सामथ्र्य दिधलें तेणें निफजे : संहारी उपक्रमीं ईश्वरईच्छा : मग शारीर मानसिक चैतन्याचा व्यापार : पाठीं जीवाचे शारीर : मायेचें सूत्र : बोधीं तयाचें शारीर : ईश्वरीचें सूत्र :।I: ॥१८॥

लापिका :- प्रपंचाते रची म्हणितले तरि कैसें रची पां? ना प्रपंच कारणभूत होता तयाचा कार्यरुप कंचुकवत् विश्वमळ रची : तयांतु रिगे तादात्मक पातलेयां आपणेयातें उघडेया देखे : ॥
वचन :- मग जीवु नाहीं तेचि एक गंधर्वनगराकार विश्व रची : तया माजि रिगे : तादात्मकु होए : विश्वमंह ऐसे बुझे ॥
टीका :- जीव म्हणितलेया एरहीं लाभे : तरि हाचि कां उच्चारिला : ना यावरि ईश्वरहेतू आणि आदिभावो जीवाचा प्रपंचरचने : यातें करितां देखे तैसें तेंही करी : आपुलिया वासना : तयावरि ईश्वरहेतू नाहीं :॥: नाहीं तेचि एक म्ह. संहारी प्रपंचाचा अभावो जाला, कारणत्वें बीजरुप होता यातें नाहीं बोलिजे : गंधर्वनगराकार म्ह. हे उपपत्ती : अर्थ सिद्धी लागीं : यथा तें दिसे सवेचि नाहीं होए : नव्हतें दिसे तैसा प्रपंच सृष्टीकाळीं नव्हता रचे : विश्व हें तियेचिया मळातें म्हणितले : देवते, पिंडस्थे, मळा एक नांव : रची म्ह. मळ घे : मागिला वचनाचेया अर्थाचा एथ विनियोग : अधमति याचा : रचीति म्ह. मति : रिगे म्ह. गति : तादात्म म्ह. रति : बुझे म्ह. मळाचें आत्मत्वें दर्शन : तें तवंचि जवं आणिकें मळीं तादात्म होए : एणेंचि क्रमें अवघें जाणावें :॥: विश्वेचा मळ तो भूतांचीं कारणें : केवळें : यथा कापुस : लोंव : तसरि : पाटसुत : ताग तें आंगी : ॥१९॥

लापिका :- तरि उघडियां देखौनि काइ करी पां? ना पूर्ववत् अष्टधा प्रकृतीमळ रची : ॥
वचन :- मग जीवु नाहीं तेचि एक गंधर्वनगराकार अष्टधा प्रकृती रची : तियेमाजी रिगे : तादात्मकु होए : प्रकृतीरहं ऐसे बुझे ॥
टीका :- मग आठांचीं भूतें तिये मागिलापासौनि समळें : कार्यरुपें : तेथौनि अन्यव्यक्ती : पृथ्वीचां ठाइं तें प्रधान सात अनुषंगिकें : ऐसें एकें भूतीं आठ : मग ऐसेया आठां एक मळ : मग ऐसेया आठां मळां रचे तो प्रकृतीमळ : प्रकृती एक एकें मळी चौसटी भूतें : प्रकृती पृथक मळीं जीव एकत्र कालवला असे : जैसी अविद्या आणि जीव : जीव देवता दोन्हीं जात्यंधें : ॥२०॥

लापिका :- तथा अनेकधा विकृतिमळ रची : ॥
वचन :- मग जीवु नाहीं तेचि एक गंधर्वनगराकार अनेकधा विकृति रची : तिये माजि रिगे : तादात्मकु होए : विकृतिरहं ऐसे बुझे ॥
टीका :- वस्त्र म्ह. विकृति : विकृति म्ह. विचित्र : मागिलापासौनि : भवन्ति यानि तानि भूतानि :॥: विकृति ते शेषशय्या ब्रह्मादिकातें म्हणितले : ॥२१॥

लापिका :- तथा अनेकधा विकृति-विकृतिमळ रची : ॥
वचन :- मग जीवु नाहीं तेचि एक गंधर्वनगराकार अनेकधा विकृति विकृति रची : तिये माजि रिगे : तादात्मकु होए : विकृति विकृतिरहं ऐसे बुझे ॥
टीका :- विकृति विकृति म्ह. विचित्रापासौनि विचित्र : कां जें उत्तरोत्तर कामावनुक धोंगडी : सामथ्र्य बुद्धी अनुरुप : कां जें निच तवं तवं धोंगडें (तोकडे) :॥: मग दोरिया दोरवेऱ्हीं : पाठाडा ते आंगी : मागिला वस्त्राचा पडिपाड विचारितां एक एकें आसुचें उणें : सेवटीं एकें आसुचें :॥: सवाए लक्ष नव कोडी : दुधाचा विकार दहीं : तयाचें ताक : मग आंब : एवं परंपरा न्यून : यें थोवे नांवे : संख्या या प्रकारें धा :॥: विकृति विकृति एणेंकरौनि इंद्रादिकां खालौतीं च्यारि स्थानें कवळिलीं : ॥२२॥

लापिका :- एवं उत्तरोत्तर मळ रचौनि सेवटील यक्षणीचां मळीं तादात्म पावे तेणें आत्यंतिक मैळे : ॥
वचन :- या परी मनुष्यपर्यंत मैळे ॥
टीका :- या परी म्ह. अनुक्रमें : मळ घेत घेत : मनुष्य म्हणितलेया कर्मभूमिचिये सेवटीली देवतेतें :॥: पर्यंत म्ह. या परता सेवट उरला नाहीं : मैळे म्ह. ज्ञान सामथ्र्य जाए : चेतनफळियाचें तेव्हेळींचि जाए : जडीचेयाचें स्थुळ रचिलेयां जाए : एरासि हा करणें नाहीं : कां जें तें एक फळचि कीं : तें तेहीं आर्जिलें नाहीं : यातवं मनुष्यमळापाठीं हाचि रची : ॥२३॥

लापिका :- हो कां : ऐसें ज्ञान सामथ्र्य गेलेयां कवण गति पां? ना माया यथाधिकारें चतुर्विध कर्मफळीं घाली : ॥
वचन :- मग जीव चतुर्विधा कर्मफळा जाति ॥
टीका :- चतुर्विधा कर्मफळां जाए तो अनुक्रमेंचि : ईश्वरासि निकेयावरि दृष्टी म्हणौनि तें अनुक्रमें घाली : प्रपंच जीवें केला : फळें माया करी : साधने तेचि दे : केवि : हें मागांचि सांगितले : लोटी म्ह. जो जेथचा तेथ कीं : मग अवघें निर्वाळलेयावरि फळीं घाली : आणि चैतन्य परंपरा मोक्षफळां जाए : यापाठीं स्वर्ग : मग कर्मभूमि : मग नरक : तेहीं सानेयांचिये ठाउनि थोर : आधींची तेथ न घलिजति : अत्यंतिक न घलितां व्यवस्था भंगैल : देवता नागवति : हा रचनेचा क्रम : संहारीं याचा विपर्याय : ॥२४॥

लापिका :- तरि चतुर्विध कर्मफळें तियें कवणें पां? ना तियें स्वर्गादिक्रमें सांगितली : ॥
वचन :- स्वर्ग : नरक : कर्मभूमि : मोक्ष यांते चतुर्विध कर्मफळ बोलिजे ॥
टीका :- स्वर्ग म्हणिजेति इंद्रादिकाचिये ठाउनि भोगभूमि पाताळीचीं मनुष्यें : नरक म्हणिजेति मनुष्यदेहाखालौतीं जेतुलीं पाप देहें : जडत्व हा सेवट :॥: कर्मभूमि म्ह. मनुष्यदेहावाचक : तरि हेंचि नांव कां नव्हेचि : जे भोगभूमिंचे लाभैल म्हणौनि : आणि सकळांची निष्पत्ती ते कर्मभूमीचि कीं :॥: मोक्ष तो त्रिविध : ब्रह्मादिकां शेषशय्यासि ऐसी प्रतिती, आम्हीं पुण्यातवं आलों परि पडों ना : आठ म्हणति, आम्हीं नित्य ईश्वरचि : वरील सुख बाहुल्य :॥: विस्मृती पैल : तो मोक्षचि कीं : केवळ सुख : जैसें जड केवळ दुःख : हें मोक्ष शब्दें बोलिजे : ॥२५॥

लापिका :- ऐसी सृष्टीरचना सांगितली तरि ते कदाचित् आनसारिखी होईल आणि काइ? ना तें पुनरुपि पूर्ववत् : अन्यथात्व एउलेंही नाहीं : ॥
वचन :- या परी मागौति जैसी असिली तैसी सृष्टी रचे ॥
टीका :- या परी म्ह. न्यूनाधिक्यातिरिक्त : यथा पूर्ववत् :॥: दुसरें : या परी म्ह. जे मागां सांगितली परी : प्रपंचाची रचना : फळाचा निर्वाळा : साधनाची प्राप्ती : मागौति म्ह. पूर्वीं होती तियेचा संहार जाला ; पुढति तेचि जीव, तोचि प्रपंच, तियेंचि फळें, तियेंचि साधनें : मागौति एणें शब्दें पूर्वील तेचि साधिली : जैसी असिली तैसी रचे म्ह. पूर्वी होती तैसीचि यथावत् रचना होए : ॥२६॥

लापिका :- ऐसी सृष्टी रचौनि माया काइ करी पां? ना ते निरंतर परमेश्वराची प्रवृत्ती वोळगतीचि असे : ॥
वचन :- माया परमेश्वराची प्रवृत्ती वोळगे ॥
टीका :- माया म्ह. ईश्वरीची व्यापारति शक्ती : स्वातंत्रिये उपजे भंगे तें प्रवृत्ती : राहे ते इच्छा :॥: वोळगे म्ह. वेचलेंपणें उल्लेखाची वाट पाहे : यथा आंगवळेकार सेवक :॥: अव्यक्तीची प्रवृत्ती जाणों म्हणितलेयाविन नेणें : व्यक्तीं मूर्तीचा स्वीकार देखणें, मनोधर्माचें जाणणें यें दोन्हीं स्वीकरिली : ॥२७॥
