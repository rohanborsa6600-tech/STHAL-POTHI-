विद्यामार्ग

लापिका :-अंतर्यागासि तथा बहिर्यागासि कारण काइ पां? ना क्रियापाक म्हणौनि आधीं भजन सांगितलें पाठीं विद्यांचें दातृत्व मांडिलें : ॥

वचनें :- एकि विद्या ॥१॥ दोनि विद्या ॥२॥ आठि विद्या ॥३॥ सवायलक्ष नवकोडी विद्या ॥४॥ बाहात्तरि कोडी विद्या ॥५॥ एकि विद्या ते ब्रह्मविद्या ॥६॥ द्विधा शांभव ॥७॥ अष्टधा शाक्तेय ॥८॥ सवायलक्ष नवकोडी महाणव ॥९॥ बाहात्तरि कोडी आणव ॥१०॥ विषयाधार वेधु ॥११॥ इंद्रियाधार वेधु ॥१२॥ अंतःकरणाधार वेधु ॥१३॥ देहाधार वेधु ॥१४॥ विदेहाधार वेधु ॥१५॥ केवळाधार वेधु ॥१६॥ विषयवेधें विषयांची प्राप्ती ॥१७॥ इंद्रियवेधें इंद्रियांची प्राप्ती ॥१८॥ अंतःकरणवेधें अंतःकरणाची प्राप्ती ॥१९॥ देहवेधें देहाची प्राप्ती ॥२०॥ विदेहवेधें विदेहाची प्राप्ती ॥२१॥ केवळवेधें कैवल्याची प्राप्ती ॥२२॥

टीका :- देवता, संचार, सामथ्र्ये, उपभोगप्राप्ती सांगितली : तयाचेनि ज्ञानें तया मोक्ष :॥: विद्यांचे पाच थोवें : नांवें यें उचितें : खालीलीतेंही शांभव म्हणिजे : प्रतिती सारिखी एक : सुखबाहुल्य दोनि : स्थिति तीनि : आणि तेतुलें दुःख नाहीं :॥: एक शुद्ध : एक समळ म्हणिजे : मूर्तीचि पाविजे : तेचि प्रकाशे : आणि कांही दुःख आति :॥: विषयाधार म्हणिजे : विषयवेधें म्हणिजे : पिंडस्थेचेनि : कां जें मळाचेनि देवतेंची प्राप्ती नाहीं :॥: विषयाधार वेध कां म्हणितला : पिंडी देवता नसे आणि विद्याधिकार तरि जीवासि असे परि देवता विद्या नेदी : कां पां : हात पावो तुटला असे : तेथची देवता गेली म्हणौनि : आधार नाहीं :I।: केवळवेधें कैवल्याची म्हणिजे : जीव अविद्यायुक्त केवळ : ईश्वर मायायुक्त केवळ : मायेपरौतें तें कैवल्य : अविद्या निरसलियां जीवही कैवल्य :।I: ॥१ते२२॥
