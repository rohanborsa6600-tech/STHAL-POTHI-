महावाक्य

वचनें :- जीवप्रपंचव्यतिरिक्त सच्चिदानंदस्वरुप परमेश्वरु एकु आति : तो सर्वशक्तीयुक्त ॥१॥ नित्य : केवळ : अनादि अविद्यायुक्त वस्तु एक आति ते जीव शब्दें बोलिजे ॥२॥ विश्वरुपाचिये ठाउनि मृत्तिकापर्यंत प्रपंचु बोलिजे ॥३॥

टीका :- ज्ञान देउनि उद्धरीति : कां जें अज्ञान-मूळ संसार तो ज्ञानें निवर्तवीति : तेंचि ज्ञान घटपट पदार्थादिकांचें नव्हे : ब्रह्मांडीचिया देवता तयांचीं साध्यें साधनें, आंतरें बाह्यें तें नव्हे : तरि परमेश्वराचें विशेषज्ञान तें मोचक : तयाचां असंभवीं शाब्द : शाब्द प्रत्यया यावेया मूर्तीवरि प्रतिती : एवं ब्रह्मविद्या : तें तयातेंचि प्रकाशीति : ऐसाचि पदार्थ : तरि अव्यक्ता देवतांचीं जीवांची ज्ञाने होतें असति कीं : तियें अनुषंगिकें म्हणौनि : सृष्टीमध्यें च्यारि पदार्थ : देवता पुढां शक्तीत्वें व्यावृत्तिजेति :॥:०:॥: तरि आधीं जीव कां : साम्य असे म्हणौनि : अव्यक्त, केवळ, देवतांचें अस्तित्व : प्रपंची हें नाहीं : तरि प्रपंच ब्रह्म परिणाम होईल :।: निर्मुळाचा विस्तार : यथा लाघवी ताटावरि आंबा दाखवी : तीन कारणें म्हणौनि दोनि बोलिलीं : मग घे सांडी तो स्थुळ : जीवीं संलग्न असे : संहारीं नासे तो सुक्ष्म बोलिजे :॥:०:॥: अविद्यायुक्त म्ह. जैसीं मसी पाणियें कालविली : पांढुरता न भंगौनि तेंचि जालें : अनादि म्ह. जैसें साळीं सडवोंडा तैचि उत्पत्ति : ॥१-३॥

वचन :- व्यतिरिक्त म्हणिजे वेगळा ॥
टीका :- वेगळा म्ह. तें दोन्हीं जेथ असति तेथ तो नाहीं : तरि अव्यापकत्व आपजैल : आणि सृष्टीमध्यें स्वरुप नाहीं : तरि अवतार उद्धरण कैसें : म्हणौनि असौनि वेगळा : यथा पिता पुत्रावेगळा : यथा उदकामध्यें चणे : ना ऐसा सृष्टी जाला तरि संबंधफळ होआवें : आणि खेदाल्हादादि संतासंते होआवीं : म्हणौनि कारणत्वें मिसळला परि कार्य नाहीं : आणि व्यापकत्व न भंगौनि तयांत शिरला असे : परि तें दोन्हीं नाहीं : यथा उदकामध्यें वस्त्र असौनि वेगळें : जीवाचा ठाइं सकळासिही मिळिजे ऐसी शक्ती असे : आणि प्रपंची हें नाहीं : पंचभौतिक त्रिगुणात्मक प्रपंच : ॥४॥

लापिका :- तरि जैसाचि ईश्वर जीवा प्रपंचा वेगळा तैसाचि ब्रह्म मायाही वेगळा म्हणों आणि काइ? म्हणें ना : ब्रह्म, माया, ईश्वर ऐसा त्र्यंश परमेश्वरु एकु आति : ॥
वचन :- सत् शब्दे ब्रह्म बोलिजे : चित् शब्दे माया बोलिजे : आनंद शब्दे ईश्वरु बोलिजे : ब्रह्म : माया : ईश्वरु ऐसा त्र्यंश परमेश्वरु एकु आति ॥
टीका :- आनंदादिकें नामें गुढें : संकेतीचीं : उपदेशांतुलें : एरें शास्त्रीं बोलिजेति प्रसिद्धें :॥: ब्रह्म अभिन्नरुप : यथा पाठी पोट :।: यथा अंगुळिएचां ठाइं अस्थि, मांस, रुधिर यें तिन्हीं असौनि एक वेगळें करुं न ये : तैसी एरें दोन्हीं वेगळीं करुं न येति : कां तें स्वरुप कीं : (माया) ईश्वरेसीं नित्यत्वें साम्य नाहीं : कां जें वेचरुप अन्यथात्व असे : आणि तो अप्रतिरुप अवघांचि अर्थीं : हे दोन्हीं पक्ष कवीश्वराचें :॥: म्हाइंभटाचा पक्ष : ईश्वरीं विरुद्धत्व असे : जैसा अपरोक्षज्ञानिया उभय प्रपंची तादात्म्य फिटलें परि देहावेगळा हों जाणे तैसा मळाहुनि नव्हे : ॥५॥

लापिका :- हो कां : ऐसा तिन्हीं मिळौनि असाधारण परमेश्वर सांगितला तरि तेथ आणिक कोण्हांचां योग नाहीं आणि काइ? म्हणें ना : तो सर्व शक्तीयुक्त कीं : ॥
वचन :- शक्ती शब्दे सकळाही देवता : सकळही गुण : सकळही धर्म : ज्ञान : विज्ञान हे सर्व शक्ती शब्दे बोलिजे ॥६॥ युक्त म्हणिजे एतत्सहीत ॥७॥
टीका :- देवता म्ह. दिव् हा धातु प्रकाशीं वर्ते : तरि प्रकाशक : आणि चेतन : आणि सामथ्र्यरुप : आल्हादकारक तें गुण : तयातें सांडुनि जाति : आणिका दिधलें होति : यथा फुलीं परिमळ : पांढुरता हा धर्म : आकार हें स्वरुप : ॥६-७॥

लापिका :- तरि जैसिया चेतनाशक्ती स्वरुपें व्यक्तामंता तैसियाचि एरीं जडाही म्हणों आणि काइ? म्हणें ना : तिया सामथ्र्यरुपा समग्र स्वरुपीं जडलेपणें असति : ॥
वचन :- सौंदर्य : लावण्य : औदार्य : सौभाग्य : दया : मया : कृपा : करुणा : एवमादिक अनंत गुण बोलिजेति ॥
टीका :- स्वरुपावरचील दैवी तें लावण्य :॥: पात्रापात्र, संभवासंभव, आवाकां पाहाणें नाहीं तें औदार्य : तोकडेया बहुत दिधलें : यथा अनुसरणे अपरोक्ष :॥: तोकडेनि गुणें बहुत आवडी तें सौभाग्य : मा ईश्वर जनप्रियु : तो गुण समुद्र कीं : गुणी आवडीं उपजे ते अज्ञानासि : ज्ञानियासि निर्गुणीं : तरि गुण तें लाभालेंखीं :॥: कृपा तीन प्रकार : दुःख फेडिजे, सुख किजे ; दोन्हीं किजति : निर्निमित्त तें कृपा : यथा कृमी ब्रह्मविद : अवघेंयां गुणा कृपा हे थोर : कां जें अव्यक्तीं देखिली : भक्ता केसणें दिधलें : ॥८॥

लापिका :- तरि जैसिया जडा आणिका दिधलेयां होति तैसेचि धर्मही म्हणो आणि काइ? म्हणे ना : तें आणिका दिधलेया न वचति : स्वरुपी अविनाभाविये असति : ॥
वचन :- नित्यत्व : व्यापकत्व : सर्वात्मकत्व : सर्वातीतत्व : सर्वकतृ©त्व : सर्वसाक्षित्व : एवमादिक अनंत धर्म बोलिजेति ॥
टीका :- व्यापकत्व सर्वात्मकत्व याचा भेद : जैसें उदकांत झाड आणि रसही तोचि : नित्यव्यापक स्वरुप : तेथ धर्में काइ केलें : जैसा तनुसाधनवंत वज्राचा आणि आंगी वज्रकवच : यथा वज्रकिटकी आगियें जळेना आणि अग्निस्तंभाची वीजु : अग्निचां ठाइं दाहकत्व : ब्राह्मणाचां ठाइं ब्राह्मणत्व : यथा (तथा) व्यापकत्व आत्मकत्व : अवघाहीं धर्मी धर्म शब्द : सर्व जेथ नाहीं तेथ आत्मकत्व कैचें : तरि स्वरुपतवं होए कीं : मग नाहीं एतुलेयाचें काइ : शक्ती परस्परें भेद : पांच थोवे, तीन आवांकें : आदिपश्चात सांगितले तत्रापि विशेष असे : कां जें ज्ञानापाठीं विज्ञान सांगितले यातवं : देवता शक्ती चेतना, भावरुपा, एकदेशिया : एरीं जडा, व्यापका, अभावरुपा : ज्ञान, विज्ञान प्रकाशक होउनि यथार्थ अन्यथार्थ : आणि व्यक्ताव्यक्तीं सारिखी सामथ्र्यरुपता तें गुणीं नाहीं : आणि तें सोडुनि (सांडुनि) जाति : एरें तें अविनाभावियें असति : गुण अव्यक्तीं कारणरुप : व्यक्तीं कार्यरुप : याचा विपर्याय धर्मीं :॥: जीव प्राप्तीं आपणयांएवढा केलेयांही परि ब्रह्मत्व नापजे : कां तें वडील होउनि व्यापक कीं : ॥९॥
