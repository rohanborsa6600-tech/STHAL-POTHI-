युगधर्म

युगधर्म

लापिका :- अगाधा, अनंता, ज्ञानानंदमयरुपा, सर्वेश्वरा, सर्वकर्तेया परमेश्वरासि जीवोद्धरणव्यसनोद्भवानंतरें परमेश्वरें कृपादिगुणशक्त्यांगिकारें चिन्मायेसि प्रवृत्ती जाणउनि ; तियेकरवी सृष्टीरचना करवुनि ; तियेमध्यें अवतरौनि उभयदृश्यत्वाचेनि स्वीकारें प्रथम लीळावशें तीन सृष्टीपर्यंत ज्ञान दिधलें : मग तेणें ज्ञानासद्भावें चतुर्विध लेपनिष्पत्तीतवं जीवार्जवें आणि परमेश्वर सर्वकर्ता तयाचिया प्रवृत्तीनुरुप जाणौनि माया तैसी तैसी सृष्टीव्यवस्था, तथा युगव्यवस्था, तथा तियें तियें युगीचें धर्माधर्म, तथा तयांचीं साध्य संपादकत्व, इत्यादि व्यवस्था मनोधर्में आकारुनि सकळ देवतासि जाणवीत असे : ऐसिये सृष्टीमध्यें परमेश्वर अवतरौनि स्वकीयें बोधदातृत्वें देवता व्यावृत्तिलिया आणि तयांचे अंतर्याग बहिर्याग व्यावृत्तिले असति : एवं युगधर्म :।: तिये युगधर्मीं प्रथम कृतयुगायुष्य, तथा तेथील धर्माधर्म, तथा दोहीचें साध्य संपादकत्व सांगितले असे : ॥

वचनें :- बत्तीस लक्ष कृतयुग ॥१॥ कृतयुगीं आत्मोपास्ती हा धर्मु ॥२॥ भक्ती अधर्मु ॥३॥ धर्मातवं चैतन्यप्राप्ती फळ ॥४॥ अधर्मातव खालीली युगीं जन्म ॥५॥

टीका :- युगधर्म म्हणिजे : जवं तें युग : आणि उत्तमा फळातें साधी : बाह्य साधनें तिये कवणें :॥: कृत म्ह. एरेंही केली : तियें मनुष्यें :॥: युग म्ह. भूमि, मनुष्यें, काळ सरेतवं :॥: आत्मोपास्ती म्ह. देह : हो कां : देहाची उपास्ती भलतेंही करी : तिया उपास्ती काइ फळें पा : तरि सूक्ष्म विशिष्टातें : तो मळ कीं : तरि आत्मा म्ह. चैतन्य : मिसळे : दोन्हीं वाती : जैसें देह आणि जीव :॥: चैतन्य उपास्ती कां : तेथ आनंद थोर : उपभोगत्वें देह द्रव्य :I।: युगधर्मे देवतांचिया मूर्ती स्वरुपांत : जैसा दिवेयाचा प्रकाश : डोळा : तेतुकालेंचि देखे : चहुंकडे : अपरोक्षीं : अंतर प्रकाश : देवता मूर्ती प्रकाश : भीतरीं ज्ञानें प्रकाश :।I: उपास्ती म्ह. उप जवळीकें : तरि आम्हीहीं असो कीं : आम्हीं काहीं देखत असों : उपास्ती म्ह. उपासनें : आत्मा म्ह. लिंगदेह : चैतन्यहीं आदि एकभेंडत्वें देखे तो : अधर्म म्ह. धर्म नव्हे तो : वांचौनि पापरुप नव्हें :।I: वरीली युगीं स्वभाव नाहीं : तज्जनित उपद्रव नाहीं : एकासि तादात्म्य नाहीं : एकासि असे : अधिकारावांचौनि धर्माधर्म न संचरति : केतुलें एक दी वाढविला : सकळ धर्माची व्यावृत्ती : ज्ञान शब्दें मुख्य भक्तीची व्यावृत्ती :।I: वरील युग सरतां तयाचां ठाइं कवण जन्मवी : तेथ उचचि जन्मवी : कां पां : खालिली रिगावा नाहीं : आणि साउमिली राहाटी चालवावी हा मनोधर्म असे म्हणौनि : ॥१-५॥

लापिका :- अधर्मातवं खालिली युगी जन्म म्हणितले तरि तें खालील युग कवण पां?

वचनें :- सोळा लक्ष त्रेत ॥६॥ त्रेती भक्ती हा धर्मु ॥७॥ यागु अधर्मु ॥८॥ धर्मातव कैलास वैकुंठप्राप्ती फळ ॥९॥ अधर्मातव खालीली युगीं जन्म ॥१०॥

टीका :- त्रेत म्ह. तिही देवतांची सत्ता म्हणौनि :॥: भक्ती म्ह. भजन लक्षण : मर्दना दे : मादने सुये : षोड्शोपचारी पूजा : गाणें : वाणें : नृत्य : आवडींपूर्वक : देहावधी :॥: मातेयापितेयाचि उत्तम : गुरुची मध्यम : वडिला बंधुची कनियस : यांत नव कोडी भेदाधर्माचे : तैसेंचि अधर्माचे :॥: नसुधियां जिवां बहिर्यागियाचां ठाइं फळचि : विद्यावंता, देवताचां ठाइं फळ होउनि अधिकारही आति विद्येचा :I।: भक्ती ते ध्यानरुप : क्रियाहीं करितां धर्मातें रामातें विसरे ना : म्हणौनि ध्यान : मूर्तीचें ज्ञान सुख संपादी : स्वरुपीचें न संपादी :।I: प्राप्ती तथा फळा भेद :॥: अर्जुनदेव स्वर्गां गेलें : भीष्मदया शाप दिधला तो जीव स्वर्गीं गेला : रेवांतक स्वर्गां गेला : शेषशायें लेंकुरुंवें आणिलीं क्षिराब्धी : एतुलेयां प्राप्ती झाली परि फळ नव्हे : संहारीं तमीं जीव असति तें प्राप्ती होए परि फळ नव्हे : अवगळां जीव असति तें प्राप्ती होए परि फळ नव्हे : देवता आणि फळिया शापिला : देवता फळियेंसि अवतरली : तें फळ होए परि प्राप्ती नव्हे : आणि अपरोक्षज्ञानियासि विघ्न : विश्व चैतन्याचें आनंद संचरले तें फळ होए परि प्राप्ती नव्हे : स्वर्गीं विद्या होए ब्रह्मेयाची तें फळ होए परि भोग नाहीं : तरि प्राप्ती होए, फळ होए, उपभोग नाहीं : म्हणौनि फळां आणि प्राप्ती भेद :I।: बहिर्यागी सद्य : प्रारब्ध : ॥६-१०॥

लापिका :- ऐसी त्रेतायुगव्यवस्था सांगितली तरि द्वापरव्यवस्था कैसी पां?

वचनें :- आठ लक्ष द्वापर ॥११॥ द्वापरी यागु हा धर्मु ॥१२॥ तीर्थ : क्षेत्र : व्रत : दान अधर्मु ॥१३॥ धर्मातव स्वर्गप्राप्ती फळ ॥१४॥ अधर्मातव खालीली युगीं जन्म ॥१५॥

टीका :- द्वापर म्ह. कृतयुगा आणि त्रेता आरुतें : दोहीसि अपर म्हणौनि द्वापर :॥: याग म्ह. देवताउद्देशे हवनें : अग्निमुखीं जो द्रव्य त्याग :॥: हा धर्म म्ह. जवं तें युग तवं हाचि धर्म : आणिका धर्माचा संचार देवता हो नेदी : धरिला म्हणौनि धर्म :॥: तीर्थ क्षेत्र अधर्म म्ह. हिने देवताचा (देवतेचा) धर्म : उत्तमें काळीं घडला तो अधर्म जाला : फळ अंतरलें :॥: खालीली युगी जन्म म्ह. तेणें अधर्में खालीली युगीचें मनुष्यदेह लाहे : कां जें असंत क्रिया तरि नव्हें : परि काळव्यतिरिकें घडली : केली क्रिया वायां न वचे म्हणौनि :॥: धर्म करितां जे हिंसा निफजली : सानियेचा फळींचि मिश्र भोग : धाडी : खेद : आल्हाद : थोर कां : पातृत्वाचा भोग तो वेगळा : आधीं असंत : पाठी संत : संपूर्ण क्रिया निफनली परि फळ न होवावया एतुलीं कारणें : लेप प्रारब्ध : समावेश नाहीं : कां कव्हणी आड रिगे : ॥११-१५॥

लापिका :- ऐसी द्वापरयुगव्यवस्था सांगितली तरि कलियुगव्यवस्था कैसी पां? ना ते सांगता असिजे : ॥

वचनें :- च्यारि लक्ष कलियुग ॥१६॥ कलियुगीं तीर्थ : क्षेत्र : व्रत : दान हा धर्मु ॥१७॥ हिंसा अधर्मु ॥१८॥ धर्मातव अंतराळप्राप्ती फळ ॥१९॥ अधर्मातव नरक ॥२०॥ उत्तमा धर्मातव अंतराळप्राप्ती फळ ॥२१॥ मध्यमा धर्मातव अष्टौदेवयोनिप्राप्ती फळ ॥२२॥ कनियसा धर्मातव अधोनिकप्राप्ती फळ ॥२३॥ कृतयुगी लक्षभरि आयुष्य ॥२४॥ त्रेतीं दहा सहस्त्र ॥२५॥ द्वापरीं सहस्त्र ॥२७॥ कलियुगी शत ॥२६॥ या चहुयुगीचां धर्मी परमेश्वरु नेणिजे न पविजे : परमेश्वरें परमेश्वरु जाणिजे : परमेश्वरें परमेश्वरु पाविजे ॥२८॥

टीका :- कळी म्ह. आकळिलें : काइ पां : शता आयुष्याचें कृमितुल्य एतुलें आकळिलें : सत्वातें धर्मातें आकळीं : आकळीं म्ह. लोपतीं :॥: युग म्ह. धर्म आणि कर्मभूमिचा योग म्हणौनि :॥: देवता-अधिष्ठान जिये उदकीं तें तीर्थ : एरें याचेंनि भावें फळ देति :॥: क्षेत्र म्ह. जवं पाला पाणी पडे तवं : तो सरलेयां केतुले एक दी वाट पाहे : देवता-अधिष्ठान व्यतिरीकें तीर्थ क्षेत्र अंतराळातें देवों न सके : एखादां देहीं अविरोधें भोगवी : यथा धानीं तांबुळ : त्रेतीं तीर्थक्षेत्रातवं खालीली युगीचें भोगविशिष्ट मनुष्यदेह होए : कां जें देवत्वातें न सधी : आणि पापक्रिया तरि नव्हे :॥: व्रत म्ह. अदृष्टबुद्धी जें नियमणें : मग एक शास्त्राचि तवं : एक भावें : स्वोक्ती : काळत्रयाची :॥: उपकार निरपेक्षता व्यतिरीकें स्वसत्ता निवृत्ती परसत्ता संपादणें तें दान :॥: हिंसा म्ह. जेणें पुढिला आपणेया दुःख :॥: नरक म्हणिजेति मनुष्यदेहाखालुते जडपर्यंत : मग मिश्रत्वाचे : तें चैतन्यफळ आदिकरौनि :I।: कळियुगीं नरक सांगितले : आणिके युगींचा अधर्म सांगितला : कळियुगीं आणिक फळ नाहीं म्हणौनि : आणि कळियुगी आसुळ विसुळ हिंसाचि : हिंसा नरक हें सकळ युगीचें सांगितले : कळियुगीं बहुत निफज :।I:०:I।: वाराणसी पैठणा समानत्व भूमिचेनि पवित्रपणें : देवता-अधिष्ठान तें तीर्थ : तीर्थ मुख्यें एरें अनुषंगिकें : एकवीरा मल्लीनाथ क्षेत्र : वाराणसी लोणार तीर्थ : दान देणें : एकादशी शिवरात्री व्रत : ब्रह्मविद्या उपदेश म्हणौनि तेणें निमित्तें सकळहीं सांडिलें :।I:०:॥: क्रमेंचि उचनिच विभाग असे : केवि : अनुष्ठिता क्रम तो उत्तम आधीं सांगिजे : पाठीं मध्यम :॥: दुसरें : तीर्थांत क्षेत्र, व्रत, दान यें तिन्हीं अनुषंगिकें घडतें असति परि तीर्थ एरां ती मध्यें न घडे : म्हणौनि तीर्थ उच : तैसीचि एरें तिन्हीं जाणावीं :॥:॥: त्रेतीं तीर्थ, क्षेत्र आति : केवि : तीर्थ माहोरीं सर्वतीर्थ : क्षेत्र एकवीरा : दान सहश्रार्जुनें दिधलें : लोकाचें मनोरथ पुरवी : तेथ तीर्थ घडे : क्षेत्र, व्रत, दान घडे : तरि खालीली युगी मनुष्यदेह लाहे : परि भोगविशिष्ट : सार्वभौम होए : ॥१६-२८॥
