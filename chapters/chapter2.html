अन्यव्यावृत्ती

अन्यव्यावृत्ती

लापिका :-सर्वात्मा, सर्वेश्वर, अपारकृपादि, सद्(षड्)गुणरत्नाकर, परमानंदामृतसागर, कैवल्यादिदानी, औदार्यराजत(राज्य)जळधर, जीवोद्धरणव्यसनिया परमेश्वर सृष्टीमध्यें अवतरौनि तेणें कर्मभूमीस्थितां मनुष्यदेहयुक्ता जीवांप्रति स्वस्वरुपादि सकळ पदार्थांचे यथातथ्ये ज्ञान केलें : कां जें ज्ञानेंविन उद्धरण नाहीं म्हणितले म्हणौनि : तरि उद्धारक ज्ञान तें कवणाचें? ना परमेश्वराचें : तरि एरें कां निरुपिली : ना व्यावृत्ती अर्थीं : कां जें अवरीचिया (अवराचिया) व्यावृत्तीविन परप्रतिष्ठा नव्हे : म्हणौनि आधीं अन्यव्यावृत्ती निरुपिली असे अनुक्रमे : एवं व्यक्ताव्यक्त निर्वचपूर्वक नवा वाक्यीं (मायादि) यक्षिणीपर्यंत सकळ देवतांची षट्प्रकारीं व्यावृत्ती केली जया पदार्था खाली तया परमेश्वर पदार्थाचिये प्रतिष्ठेअर्थीं दाहावें वाक्य निरुपिलें : तेचि प्रथम वचन पाठ : तयाचा अर्थ आठा वचनीं ऐसें नऊ : दाहावें परमेश्वरप्रतिष्ठा : ॥
वचन :- कर्मभूमीचिया देवता होति परि परमेश्वरु नव्हति ॥
टीका :- सेवटिली यक्षणीचें स्वरुप तवं कर्मभूमी एवढें होआवेंचि : केवि : जें पिंडीं अवघियां होआविया लागति : भोगभूमीचे पिंड ते ईश्वरवत् : स्वर्गीं अवतरले तरि एथीचिया नाहीं : यथा अर्जुनदेव क्षीराब्धी(क्षीराब्धा) गेले एथीचिया राहिलिया : देवते आपणेयांखालौते ऐश्वर्य असे : परि भलें तें नाहीं : बोध तथा शक्ती : तनुसाधन : कर्मलेप नासैल काइ : परि कोरभरी उणाहीं नव्हवे : यथा लक्ष्मणें रामाची सेवा केली : अविद्येचा नाश एवमादिक :॥: कर्मभूमी ऐश्वर्य असे : म्हणौनि परत्व : परि परमेश्वरत्व नाहीं :I: कर्मभूमीचिया देवतास्वरुपाचा उचनिच विभाग नेणिजे : फळियांसि कर्माचें ज्ञान नाहीं : मी पडेन हा धाक : इंद्रादिकाचिये ठाउनि खालौता :।I:०:॥: भोगभूमी मनुष्यें असति केवि : कर्मभूमीस्थिता मनुष्यदेहेयुक्तां म्हणौनि : आणि पांच शतें कर्मभूमी म्हणौनि : आणि कर्मभूमी एक करितां एक निमे एक निफजे म्हणौनि : आणि संबंधातवं भोगभूमी मनुष्यदेह होए म्हणौनि : आणि भोगभूमी अत्रीचां ठाइं अवतार स्वीकरिला : म्हणौनि भोगभूमी मनुष्यें असति :॥:०:I।: कर्मभूमीचिया देवता जळीं : विहीरी : राउळगणीं : दिव्यभोग : इंद्रादिकांचिया खालौति दोघदोघे : विषयसंबंध स्वल्प : भावीं रमण बहुत : यागाचेनि कनियेसपणें प्राप्ती :।I: ॥१॥

लापिका :- ऐसी (ऐसिया) कर्मभूमीचिया देवतांची व्यवस्था सांगितली जरि तरि तयां वरिष्ठे देवतांची कैसी पां? ना तियाही तेराचि कोडी असति आणि स्वरुप-सपुरत्वेंही मागीलाचि सारिखियाचि एकी पस्से एकी श्रेष्ठाचि असति : तथा पंचप्रकारेंही श्रेष्ठाचि असति :।: मा तरि तयांची अव्यक्त स्वरुपमर्यादा काइ पां? ना ते कर्मभूमीचिया देवतास्वरुपापस्से दशांशे अधिक असे : एवं पांच सहस्त्रेचि सकळांची स्वरुपावधी असे : मग तयांचे विभाग तथा पंचप्रकार ते आपुलालिये स्वरुपावधी आंतुचि कार्यत्वे वर्तति परि तयांचिया ब्रह्मांडस्थमूर्तीचे आठचि प्रकार होआवे ऐसी स्वरुपसमृद्धी असे : तथा तया ब्रह्मांडस्थासि रचनारुप स्थान नाहीं आणि निएतवासे असावें ऐसें नाहीं : म्हणौनि तयांसि स्थान नाम उच्चारिलें नसे : ॥
वचन :- अष्टौदेवयोनी देवता होति परि परमेश्वरु नव्हति ॥
टीका :- अष्टौ म्हणिजेति आठ प्रकार : एकी जाति होवोनि विलक्षण : यथा घोडीं गाढवें : गाइ म्हैसी : सेळिया मेंढिया :॥: योनि म्हणिजेति नसुधिया मूर्ती : एथ येति : भोगभूमी जाति : आवडे तेथ असति : एक स्थान नाहीं : यथा भाटीचे वसो गल्हांट :॥: अष्टौदेवयोनीसि एक स्थान नाहीं : कां गावाचें नांव नाहीं म्हणौनि : अष्टभैरवांसि विश्वेसि स्थान नाहीं तैसें अष्टौदेवयोनीसि नाहीं : एरां देवतांचिया स्थानाचे नांव घेतलें :I।: अष्टौदेवयोनीसि एक स्थान सांगितले नाहीं : तेहीं अंतरिक्ष : भाटायोनि म्ह. (म्हणिजेति) घोडे : ईश्वर आपणेयातें नियेमीतिना : परि कृपा नियेमिलें : यथा ब्राह्मणातें ब्राह्मण्य नियेमी :।I: ॥२॥

लापिका :- ऐसा अष्टौदेवयोनीचा दुसरा थोवा तयाची व्यावृत्ती ऐसी सांगितली जरि तरि तिसरा थोवा गंधर्व : तयाची व्यवस्था कैसी पां? ना गणना तथा स्वरुप-सपुरत्व तें अष्टभेदीं : तथा स्वरुप-मर्यादा तथा पंचप्रकार तेहीं दशांशेचि असति : आणि ब्रह्मांडस्थासि पृथ्वी संबंधिक रचनारुप स्थान नसे : ॥
वचन :- अंतराळीचे गंधर्व देवता होति परि परमेश्वरु नव्हति ॥
टीका :- वरीलांही नाहीं : परि आळय म्हणिजे : विमान : जुंगवत् : ॥३॥

लापिका :- ऐसी तेरा तिरी (तिन तेरा) एकोणचाळीस कोडी देवतांची व्यावृत्ती केली जरि तरि तयापरौतें चवथे स्थान तें मेरुचा पाठारी दिव्य दिप्तायमान रचनारुप असे आणि तेथीचिया देवता गंधर्वांचें गायनही ऐकति असति : तरि तयाचिया स्थानाचे नांव स्वर्ग : ॥
वचन :- स्वर्गीचे इंद्रचंद्रादिक देवता होति परि परमेश्वरु नव्हति ॥
टीका :- इंद्र म्हणितलेया चंद्र सूर्य मुख्य : परिवार आदिक या शब्दांत लाभे :॥: स्वर्ग हे स्थाना नांव : एऱ्हवीं मनुष्यदेहावरौतें कर्मभूमीपासौनि तो ठावोवेऱ्हीं स्वर्गचि : तें फळावाचक :॥: देवता म्ह. दिव् हा धातु प्रकाशीं वर्ते : चेतनत्व, सामथ्र्य, प्रकाशरुपता, ज्ञान, सुख, ऐश्वर्य असे : परि परमत्व नाहीं : म्हणौनि नव्हति : ॥४॥

लापिका :- ऐसें स्वर्गस्था देवतांचां ठाइं षट्प्रकारे लक्षण सांगितलें एणेंकरौनि वरौतें स्थान कैलास वैकुंठ तेथ विशेषें देवतात्व असे परि परमत्व नाहीं : ॥
वचन :- कैलास वैकुंठीचे हरिहरब्रह्मादिक देवता होति परि परमेश्वरु नव्हति ॥
टीका :- वैकुंठ स्थानें दोनि : देवता तीनि : ततश्च एरहीं जाणावें : कां पां : ग्रामराणे म्हणितले म्हणौनि : ॥५॥

लापिका :- ऐसें तिन्हीं देवतांचां ठाइं षट्प्रकार समानत्व सांगितले जरि तरि तयापरौतें साहावें स्थान तेथील देवता तयांचे लक्षण तथा स्थान व्यवस्था कैसी असे पां? ना ते विलक्षणचि जाणावी : ॥
वचन :- क्षीराब्धीचा शेषशय्या देवता होए परि परमेश्वरु नव्हे ॥
टीका :- शेष म्ह. सर्पाकार : कल्पतरु म्ह. वृक्षाकार : तेथही पशू तिर्यंचें : एक देवता : एक जीव : पार्यातकवत् :॥: गावा नांव क्षिराब्धी : जैसा ढोर समुद्र : तेथ काइ ढोरें असति : ॥६॥

लापिका :- हो कां : ऐसी क्षिराब्धीची व्यवस्था सांगितली तरि सातवें स्थान भैरव तयासि स्थान नियम नसे : तथा स्थिरत्व नसे आणि अहंकार-कार्ये तरि बाहुल्यें वर्तति असे : ॥
वचन :- अष्टभैरव देवता होति परि परमेश्वरु नव्हति ॥
टीका :- सृष्टीकर्ते : माझी सृष्टी ऐसा वेच असे : परि नाहीं : म्हणौनि अंतररमण कीं :॥: एसणे मोक्षपद पावौनि बाह्यरमण : तेंचि कवण पां : ना सृष्टी : संहार : अवतार : अवतरण : हे विश्वाही आति कीं : ते सकृत : तेथ काइ उर्मी असे : यथा संहारवत् :॥:०:॥: अष्ट म्हणिजेति आठचि : परिवार नाहीं : एक स्थान भूमि असे : परि ठाइं ठेले नव्हति : चंचलता : सवेचि ऐऱ्हा : भैरव म्हणिजेति अहंकार उर्मी बाहुल्यें : ॥७॥

लापिका :- भैरवव्यवस्था ऐसी सांगितली जरि तरि तयापस्से श्रेष्ठ देवता विश्व : तरि तियेचें श्रेष्ठत्व कैसें? ना तियेचे अव्यक्त स्वरुप तरि पांचशे कोडी असे : आणि तियेची ब्रह्मांडस्था तरि ब्रह्मांडाएसणी थोर : एवं छपन्न कोडी : तियेचा गर्भावकासीं चराचर विश्व सामावले असे : ॥
वचन :- विश्वरुप देवता होए परि परमेश्वरु नव्हे ॥
टीका :- विश्वाचिये ठाउनि प्रपंच : विश्व जयाचा रुपीं : रुप म्ह. अव्यक्त : ॥८॥

लापिका :- हो कां : विश्वरुप देवता होए परि परमेश्वर नव्हे म्हणौनि विश्व व्यावृत्तिली जरि तरि तियेपसि सकळाअर्थी अधिक श्रेष्ठत्व मायेचा ठाइं असे : तरि तियेचां ठाइं परमत्व कल्पैल आणि काइ? म्हणे ना : ॥
वचन :- चैतन्य देवता होए परि परमेश्वरु नव्हे ॥
टीका :- चैतन्य पर होए : ब्रह्म ईश्वर होए : परि परमत्व म्हणों न ये :॥: ब्रह्म कां नव्हे : ना तया पर उरलें असे : एऱ्हवीं एकाहुनी एक पर असे : उद्धरण नाहीं : ॥९॥

लापिका :- ऐसिया चैतन्यादि सकळ देवता तयांचां ठाइं परमत्व नाहीं म्हणौनि व्यावृत्तिलिया : तरि पर (परम/परमेश्वर) म्ह. जीवाचा ईश्वर कर्ता तो कवण पां? ना चैतन्यापर परमेश्वरु श्रीचक्रधरु : म्हणौनि तोचि भटोबासीं विशेषें प्रसंसिला असे तोचि जाणावा : ॥
वचन :- चैतन्यापर परमेश्वरु तो देवो : श्रीचक्रधरु : सकळ जीवांतें रक्षिता हो : उद्धरिता हो ॥
टीका :- चैतन्यापर म्हणितलां ब्रह्मासि कां न म्हणतीचि : ना मान्यता भंगैल : कां तें वडील : वृद्ध : अगाध : आपणयापसि मानिती थोरपणें : निर्गुण म्हणौनि :॥: श्रीचक्रधर म्ह. ज्ञानचक्र शक्तीचक्र धरिलें म्हणौनि : रक्षिता म्ह. दुःखावेगळें करिता : उद्धरिता म्ह. सकळ देवता उच करुनि मोक्ष देता :॥: हो म्ह. भट ऐसें असंसीति असति : श्रीचक्रधर एथौनि भटांचे संबोधन : ॥१०॥
