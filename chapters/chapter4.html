क्रियापाक

उत्तम सत्व : उत्तमें द्रव्यें : उत्तमें पात्रें : उत्तम काळ : उत्तम पुरुष :॥: पुरुष सात्विक : पात्र सात्विक : करितीं मनुष्यें सात्विकें : स्थान सात्विक : काळ सात्विक : जडपदार्थ पर्येवसान किजति तें सात्विक : तरि केवळ सत्वरुप क्रिया निफजे : तैसीचि प्रकृती आरंभे :॥: पुरुष, पात्र, व्यापार करितीं मनुष्यें सात्विकें : स्थान, काळ, जडपदार्थ पर्येवसान किजति एतुली राजसें : तरि सत्वरज ऐसी क्रिया निफजे : तैसीचि प्रकृती आरंभे :॥: पुरुष, पात्र, मनुष्यें सात्विकें : स्थान, काळ, द्रव्यें एतुली तामसें : तरि सत्वतम ऐसी क्रिया निफजे : तैसीचि प्रकृती होए : सत्व तामसा देवतांचीं फळें लाहे :॥: पुरुष, पात्र, मनुष्य राजसें : स्थान, काळ, द्रव्ये राजसें : तरि केवळ रजरुप क्रिया निफजे : तैसीचि प्रकृती होए : राजसा देवतांचें फळ होए :॥: पुरुष, पात्र, व्यापार करितीं मनुष्यें : दासी, बटुक, चाटें, ब्राह्मणबाइया एतुली राजसें : स्थान, काळ, अन्नें, खाजीं ऐसें जडपदार्थ सात्विक : तरि रजसत्व ऐसी क्रिया निफजे : तैसीचि प्रकृती होए : आणि परलोकीं देवतांचीं फळें ऐसींचि होति :॥: पुरुष, पात्र, मनुष्यें राजसें : स्थान, काळ, जडपदार्थ तामस : ऐसें होए तरि रजतम ऐसी क्रिया निफजे : तैसीचि प्रकृती होए : ऐसियेचि देवतेचें फळ होए :॥: पुरुष, पात्र, स्थान, काळ, द्रव्यें, करितीं मनुष्ये तामसें : तरि केवळ तमोरुप क्रिया निफजे : तैसीचि प्रकृती आरंभे : तैसियाचि तामसा देवतांची फळें होति :॥: पुरुष, पात्र, मनुष्यें तामसें : स्थान, काळ, पदार्थ राजस : तरि तमरज ऐसी क्रिया निफजे : तैसीचि प्रकृती होए : ऐसियेचि देवतेचें फळ होए :॥: पुरुष, पात्र, करीति मनुष्यें तामसें : स्थान, काळ, पदार्थ सात्विक : तरि तमसत्व ऐसी क्रिया निफजे : तैसीचि प्रकृती निफजे : आणि तमसत्व ऐसियेचि देवतेंचें फळ पावे :॥:
