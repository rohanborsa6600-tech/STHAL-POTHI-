संहार

लापिका :-अनंतें परमेश्वरें स्वयमेव स्वरुपउल्लेखें कृपांगिकारें मायेसि प्रवृत्ती जाणउनि ; सृष्टीरचना कतृ©त्व करउनि ; तियेमध्यें अवतरौनि सकळ जीवां ज्ञानदातृत्व जाणिवेतवं प्रेमोपाय करउनि प्रेम देति : तेणें स्वानुभूत साध्यें पूर्णानंद भोगी : आणि एके ज्ञानासद्भावातवं ; एवं प्रमाद पतितत्वें नित्यार्जकत्वें नित्यभोग पावले : ऐसी सकळ जीवांची ज्ञानावधी जालियां नित्यनाशार्थ हेतू मायेसि जाणवीति : एवं दश प्रकारीं जीव देवता प्रपंचाचिये सृष्टीचा संहार करवीति : ॥
वचन :- परमेश्वरप्रवृत्ती माया सृष्टीसंहार करी ॥
टीका :- परमेश्वर म्हणितलेयां व्यक्त नव्हे : अव्यक्त : जेथ तिन्हीं अंश तेथ हे नाम : एर तें कैवल्य : केवळ जेथ मायासंबंध तेथचि तीहीं प्रवृत्तींचा उज्रंभ :।: प्रवृत्ती म्ह. स्वातंत्रिये उपजे भंगे :।: अव्यक्तीची प्रवृत्ती जाणों म्हणितलेयाविन नेणें : व्यक्तीं मूर्तीचा स्वीकार देखणें, मनोधर्माचें जाणणें हे दोन्हीं स्वीकरिली :॥: नसुधी माया सांगतां तिये स्वातंत्री आपजैल : ते पतिव्रता :।: दुसरें : ईश्वरां उपजे तैचि होए : म्हणौनि प्रवृत्ती :।: सृष्टी म्ह. सृजिली होती म्हणौनि : तेचि सृष्टी जीव देवता प्रपंचाची :।: संहार म्ह. रचना मोडें : संहरतीति संहार : ईश्वरीची प्रवृत्ती : मायेचा व्यापार : जीवाचे करण : प्रपंचाची रचना : कर्माची निष्पत्ती : फळींचा भोग : देवतांचे विग्रह : साधनें : अवतार : उद्धरण : एतुकां मिळौनि सृष्टी : हें अवघें नाशे तो संहार :॥: करी म्हणितलेया क्रिया न लगे : सामथ्र्यें होए : एक योगाग्नि : यथा द्वापरवत् : दुसरा युगाग्नि युगांतीचा : तिसरा प्रलयाग्नि : तो प्रगटे : प्रपंचाचा अभावो होए : तो अनुक्रमें (अनुक्रम) सांगता असिजे : ॥१॥

लापिका :- मा तरि निचादि क्रमें मळनाश सांगितला, तरि तेंचि निच मळ कवण? ना इंद्रादिकापासौनि खालौते चौ स्थानींचे : कां जें चहुतें विखो म्हणितलें : ॥
वचन :- विखो इंद्रियीं विलया जाए ॥
टीका :- इंद्रादिकांचां फळीं प्राधान्य विषयो : अनुषंगिक आनंद : देवता याचा विपर्यावो : यथा भातवनी ठिकवनी : यें योग नामें : उपभोग तो एथचियाचि सारिखा : परि तो भावें : एणें योगें मळासि नाम : तेंचि तेयाचिया ज्ञानासि आधार : वशिष्ट शुद्धविद्यावंत : एसणा आनंद परि पुत्रशोक : यथा रामदेवो :॥: अर्थप्रकाशक तें इंद्रिये : इंद्रिये म्ह. प्रकाशरुपता ज्ञान : पांचाही इंद्रियां एणेंचि योगे नाम : तादात्म्य निवर्तौनि आत्मस्वरुप प्रकाशे मळाधार : आणि आपुलिया स्वरुपाभीतरी जेतुलें जीव जड चेतन तेतुलेयांचें मळाधार ज्ञान :॥: इंद्रादिकां आपणेयाखालौतें परमाणुपर्यंत ज्ञान : घटपटाचेनि वेवधानें भंग नाहीं :॥: तरि वरीलांचां ठाइं विषयपदार्थ देखत असों कीं : तो खेळ : वांचौनि भोग तो आनंद :॥: विलयो म्ह. अत्यंत अभावो : यथा दिवा फुंकीलीयां (प्रकाश) नाहीं जाला : ॥२॥

लापिका :- हो कां : विखोमळ इंद्रियीं विलया जाए ऐसें म्हणितले जरि तरि इंद्रियें नांमे जे मळ ते कोठे विलया जाति पां? ना अंतःकरणाचां ठाइं : ॥
वचन :- इंद्रिये अंतःकरणी विलया जाए ॥
टीका :- इंद्रियांत शेषशय्याही लाभे : कां जें प्रतिती एकसारिखी : मागौते इंद्रियें : कार्य कारण साधावेया :॥: अंतःकरण वरीलां आठातें : अंतरीं आनंद : इंद्रियेंद्वारें नाहीं : विलयो म्ह. अभावो : ॥३॥

लापिका :- तथाचि अंतःकरण देहाचा ठाइं विलया जाए : ॥
वचन :- अंतःकरण देहीं विलया जाए ॥
टीका :- अंतःकरण पुढति मागिलाचि परी अवघें जाणावें :॥: देह म्ह. अवघेंया देहासि आदि : एकीकडुनि तरि सेवट : विलयो म्ह. पूर्वप्रतिभिज्ञा :॥: जाए म्ह. आदिपश्चात ना : दोहीं एकचि काळ : सानें थोर एकेंचि वेळें : यथा डोंगरीं वणवा : ॥४॥

लापिका :- हो कां : तरि अंतःकरण जें आठांचे मळ तें देह जो विश्वेचा मळ तेथ विलया जाति म्हणितलें तरि देह जो विश्वेचा मळ तो कोठें विलया जाए पां? ना तो विदेहीं विलया जाए : ॥
वचन :- देह विदेहीं विलया जाए ॥
टीका :- विदेह तियेचा मळ : अवघें एक प्रकरण : तयाचां ठाइं जाए ऐसें न बोलिजे : उच्चारक्रम ऐसा परि अर्थ नव्हे : यथा वृक्ष जळीन्नला परि बुढासकट : ॥५॥

लापिका :- हो कां : पूर्वोक्त मळनाश सांगितला जरि तरि विदेहमळ काइ पां? ना तयासि काहीं काळ लागे : मग नासें : ॥
वचन :- विदेह ते नित्य ॥
टीका :- नित्य म्ह. मागिलापस्ये या काळ लागे : यथा वोलें काष्ट : एरीं (एर/एरें) यथा शुष्कवत् : ॥६॥

लापिका :- हो कां : मागां थोवे वाक्यें (भेदे) कार्य कारण संबंध मळाचा एथपर्यंत सांगितला : आतां थोवेयाचि मध्यें पृथकाभेदीं कार्य कारण संबंध सांगताति : ॥
वचन :- कार्यअविद्या कारणअविद्येचां ठाइं विलया जाए ॥
टीका :- अविद्या म्ह. चिन्ह : जेणें आधारें देवतें वोळखी जीवाची : कार्य म्ह. जयापासौनि जें होए : यथा माया(कार्य) जीव : जीव(कार्य) प्रपंच : कारण म्ह. एरासि आदी म्हणौनि :।: उच मळ निचां कारण : मागा थोवें नावें सांगितली परि तें एणेंचि क्रमें : दोहीं एकें वेळें नास : ॥७॥

लापिका :- हो कां : ऐसा प्रपंच विषइं कार्य कारणाचा नास सांगितला तरि एर पदार्थ काइ पां? ना तेही कारणाप्रति जाति : ॥
वचन :- कार्य कारणीं विलया जाए ॥
टीका :- कार्य कारणी विलया जाए म्ह. मागिलां वचनीं पृथकाकार सांगितला, तो अवघेंयाचा यावांका, यें वचनीं समुच्चाकार : ॥८॥

लापिका :- हो कां : कार्य कारणीं विलया जाए म्हणितलें तरि नित्यानित्य संहारति आणि काइ म्हणौनि दोन वर्ग म्हणितलें : ॥
वचन :- एकु नित्य वर्गु : एकु अनित्य वर्गु ॥
टीका :- तरि जें जें अनित्य तें तें नासैल कीं म्हणौनि दोनि वर्ग सांगितले :।: काळें उपायें न नसे तो नित्य : एहीं दोहीं नासे तो अनित्य : ॥९॥

लापिका :- हो कां : ऐसें नित्यानित्य पदार्थाचें ज्ञान केलें जरि तरि नित्य वर्गींचे पदार्थ कवण? किती असति पां? ना तीन पदार्थ : ॥
वचन :- नित्य वर्गी जीव : देवता : परमेश्वरु उरेति ॥
टीका :- नित्य वर्गींचें तीन पदार्थ : जीव, देवता, परमेश्वर : जीवातें काहीचि नाहीं : नसुधा स्वस्वरुपेसि उरे : ईश्वरीं सदा उरी : देवता, ज्ञान, सुख, सामथ्र्य एणेंसी उरेति : खेदाल्हाद तें प्रपंच संबंधे : अनित्यवत् : ॥१०॥

लापिका :- मा तरि नित्य वर्गीचें पदार्थ कैसें उरेति ऐसें सांगितले तरि अनित्य वर्गींचे काइ पां? ना एक नासे : दोनि उरेति : ॥
वचन :- अनित्य वर्गी अनादि अविद्या कर्मलेप उरेति ॥
टीका :- दुसरा वर्गींचे तीन : एक नासला : दोंसि उरी :॥: उरावेयाचें कारण : एक ईश्वरैक नास्य : एक भोगैक नास्य : आणि लेप अवघें नासले तरि रचलेया कर्म फळा कैसें जाति : ॥११॥

लापिका :- हो कां : नित्यानित्य वर्ग उरेति म्हणितले तरि ते वेगळालेचि असति आणि काइ? ना अविद्या, कर्मलेप आणि जीव युक्तत्वें असे : ॥
वचन :- अनादि अविद्या कर्मलेपयुक्त जीव मायास्वरुपी असति ॥
टीका :- दो वर्गींचे दोन थोवे आपजती म्हणौनि युक्तत्व सांगितलें जीवाचें : ॥१२॥

लापिका :- हो कां : ऐसी जीवाची व्यवस्था सांगितली तरि देवतांची काइ पां? ना तिया कारणाप्रति जाति : ॥
वचन :- देवता स्वरुपी स्थिता होति ॥
टीका :- स्वरुपस्थिता म्हणिजेति यथा राउळीचीं मनुष्यें : एरी ते स्त्री रात्रींही जागतीचि असे : ॥१३॥

लापिका :- हो कां : ऐसी देवतांची व्यवस्था सांगितली तरि प्रपंच काइ पां? ना तो कार्यरुप तेतुला नासे : ॥
वचन :- या परी प्रपंचु संहारे ॥
टीका :- या परी म्ह. मागां सांगितली ते :॥: प्रपंच म्ह. जड : मिथ्यारुप : नव्हता : करणियेचा : मैळिता म्हणौनि : संहार म्ह. अभावो : नास : ॥१४॥

लापिका :- मां तरि पूर्वोक्त वाक्यें संहार जालेयां माया काइ करी पां? ना निरंतर प्रवृत्ती वोळगतीचि असे : ॥
वचन :- माया परमेश्वराची प्रवृत्ती वोळगे ॥
टीका :- माया म्ह. व्यापारती ईश्वरीची मुदल शक्ती : मोहवी : भ्रमवी : अन्यथाज्ञान उपजवी : गर्भीं वाहें : वियें : म्हणौनि माया :॥: प्रवृत्ती म्ह. उल्लेख : वोळगे म्ह. तयाची वाट पाहे : वेचलेपणें : यथा आंगवळेकार : पतिव्रतावत् : ॥१५॥
