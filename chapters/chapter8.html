उद्धरण

लापिका :-सकळ सामथ्र्य परिपूर्ण, सर्वज्ञ, सर्वेश्वर, अपार कृपादि सद्गुणसागर, जीवोद्धरणव्यसनिया परमेश्वर अनादि अविद्यायुक्तां जीवासि कैसेनि उद्धरीति पां? ना सृष्टी रचौनि तियेमध्यें कर्मभूमि अवतरौनि उद्धरीति : ॥
वचन :- सृष्टीमध्ये परमेश्वरु अवतरेति ॥
टीका :- चैतन्य मृत्तिका हा सकळ फळाचा सेवट : सुख दुःखत्वें साम्य : केवळ : मग उत्तरोत्तर (उतरत-उतरत) दोहींमध्यें कर्मभूमि : मनुष्यदेहीं अवतरण : जैसी मनुष्यघटीं देवता अवतरे : तयासि विधान सांगे : सेवा, दास्य, भजन : संसाररोगी देहाची बळी : विषय त्याग : हें सांडणे :॥: सृष्टीचा मध्य कर्मभूमि : केवि पां : जें अवघेयां ब्रह्मांड गोळकाचेया अवकाशाचा मध्य : देवतांचेया स्वरुपांचा मध्य : आणि देव, मनुष्य, तिर्यंच तो मध्य : आणि कर्मभूमिवांचौनि सकळ देवता नाहीं : ईश्वरीं नाहीं : पिंडी नाहीं : विद्येचें संपूर्णत्व नाहीं म्हणौनि मध्य : गोसावियांपासौनि लक्षांतरीं वेध म्हणौनि भोगभूमि सिल्पा होति : परि साधननिष्पत्ति नाहीं : म्हणौनि कर्मभूमिवांचौनि नाहीं :॥: परमेश्वर म्ह. परम ऐश्वर्य आथिला : तरि परम म्ह. काइ : ना उद्धरण : अवरदृश्ये, परदृश्ये, देव, तिर्यंच ईश्वर होति परि परम नव्हेति : तो उभयदृश्याचि होए : अवतरे म्ह. रुपासि ये : व्यक्त होए : यथा प्रकृती (प्रवृत्ती) :॥:०:I।: सृष्टीसंहाराचा : प्रपंचा बाहिरीला स्वरुपा समीप : अवतार : तो देहाबाहिरीला : स्वरुपाचा धर्म भीतरी अवतरे : कां देहाभीतरीचि धर्म उपजे : अवतारही स्वीकरीति : अव्यक्तौनि स्वीकरीति :॥: ईश्वरस्वरुपापासौनि समीप मायास्वरुप ते पुर होए : पुरावेगळे तें दास्य करी :॥: पुरापासि अव्यक्ता देवता तिया ब्रह्मांडीचिया सारिखिया आकारति :I।: कृपारुप धर्म ईश्वरीचि वर्तति : स्वरुपीं लोपति : क्षेदाल्हाद धर्म हे देवतांचे स्वीकरीति : जवं मूर्ती तवं असति : मग जाति : स्वभावादिक धर्म तें मनुष्याचे स्वीकरीति : तें जाति मूर्ती असतांचि : जीवाचेनि भजनें आल्हाद उपजे : आल्हादें कृपा उपजे : ॥१॥

लापिका :- तरि सृष्टीमध्यें विचित्र अवतार असति कीं? ना तें उद्धारक नव्हति : म्हणौनि आधीं युक्त मानवतनु धरौनि तेथीचे (तेथचे) सकळही गुण धर्म स्वीकरीति : ॥
वचन :- मनुष्यवेषधारीये होति ॥
टीका :- मनुष्यवेष म्ह. तयाचे गुणधर्म अंगीकरीति : तेचि हर्ष : विषाद : विसर : देहद्वंद्वे : मनोद्वंद्वे : तोष : खंति : तरि खंति कां : ना तें गुणाचि कारणें : तेणेंविन जीव इष्टीं योजतिना : हें भय जरि नसतें तरि जीवासि आज्ञाभंग करिता असळगै, पातलेपण येतां वेळुचि न लगे : म्हणौनि जीवधर्म अंगीकरिलें :॥: वेष म्ह. लटिका : स्वरुपाचें ज्ञान असौनि रुपांतरातें धरीं तो वेष :।: यथा बहुरुपी :॥: लटिका होउनि परि ईश्वरीचेनि अंगीकारें जीवासि साच : तयाचेनि भजनें, सेवादास्यें गोमटें होत असे : तो अव्यक्त केवि जोडे : या कारणें मूर्तीधार्या होउनि आपण पात्र जाला : तें पात्रतेचें फळ माया दे : कां तो अविक्रियु : तृप्तकाम : काइसेनि तोखवावा : म्हणौनि देहिक स्वभाव अंगीकरीति : तहान : भूक :॥: धार्ये म्ह. यथा वस्त्र धरिजे : आहाच जडलेपण नाहीं :॥: होति म्ह. जैसा जीव देहांत व्यापला असे तेतुला ठाइं आपण होति :॥: दुसरा ; मनुष्यवेष तो ऐसेयातें म्हणिजे : वय, रुप, गुण, उपगुण, कुशळता, चातुर्य, जाणीव, सिहाणीव, बळ, पराक्रम ; बहिर्यागाचा अंगीकार एतुलें वर्ते : ॥२॥

लापिका :- तरि काइ मनुष्यत्वचि स्वीकरीति? ना देवतात्व अंगीकरुनि चिदादि सकळ देवतांचे गुणधर्म तथा विद्याही स्वीकरीति : ॥
वचन :- मायावेषधारिये होति ॥
टीका :- मायावेष म्ह. तें मायादेह कीं : तया भजन तयाचां पदार्थीं : मां तो ईश्वर अविक्रियु :॥: दुसरा मायावेष म्ह. मायेचिये अंगीये मायेचे गुणधर्म अंगीकरीति : तरि धर्म वेगळे नव्हति कीं : ना तया सारिखे अंगीकरीति : कां अन्यत्र आति तें ईश्वरी आति म्हणौनि :॥: मायेचां धर्मी सकळ फळें, साधनें देणें : साधननिष्पत्ती करणें, साधनवंताचा अभिमान घेणें : आणि तिन्हीं दानें मायेचां धर्मीं : तिर्यंच अवतारीं एकीचि लीळा : अवरदृश्यीं तीन : कैवल्य नाहीं : परदृश्यीं अडैच : लीळा, संबंध, कैवल्य अर्ध : कां जें साधन नेदीति : साध्य देति : घेउनि देणे आति ना : कां जें कर्मलेप देखणें नाहीं : देखावें मग घेयावें कीं : ना हें वर्ते तरि आउठ : उभयदर्शीं च्यारि : परि तो पुरांत : परदृश्या तयांतुचि मोडे :॥: अवरदृश्या माया वेषांत : हें मनुष्याचां ठाइं न वर्ते म्हणौनि आपणहीं देवता जाले : मायेचे गुणधर्म सांगितलेया सकळ देवतांचे सांगितले होति : कां जें थोरांत सानें मोडे : यथा चिंतामणीमध्यें सकळ द्रव्यें : यक्षणीचेनि धर्में चैतन्यविद्या न देववे : मायेचां धर्मीं खालुतें अवघें देववें :॥: दुसरा ; मायावेष तो ऐसेया वाचक : चैतन्यविद्या मुख्य अवरा विद्यांचा अंगीकार : अविद्येचें दर्शन : सकळ जीवाचा मोहो : ॥३॥

लापिका :- तरि काइ हेचि स्वीकरीति? ना मायापुरांगिकारे सकळ स्वपर गुणधर्म अंगीकरुनि उभयदृश्यत्व परमेश्वरत्व स्वीकरीति : ॥
वचन :- मायापुरधारिये होति ॥
टीका :- मायापुर म्ह. अवघें पुरतें : तेथ काहीं उणें नाहीं : देहधर्म : जीवधर्म : मनुष्याचे धर्म : देवतांचे धर्म : मायेचे धर्म : परावर शक्तीचा अंगीकार : स्वरुपींचे गुण : जडा चेतना शक्ती : च्यारि दानें : यातें पुर बोलिजे :॥: पुर म्ह. प्रकाशक रायाचें : रावो आलेयां उभवणी :॥: धारिये म्ह. यथा आंगी धरिली : यथा घटें उदक धरिलें : घागरी भरुनि तळेयांतु चालविजे : तेंचि उदक : पाणियावरि बुडडा चाले : तोचि वायो : तैसें मूर्तीआंत स्वरुप असे : मग ते मूर्ती व्यापकांत चाले : एऱ्हवीं घटाकारत्व होईल : ॥४॥

लापिका :- तरि तें पुर कवणें निर्मिलें पां? ना स्वयंमेव माया निर्मिलें : ॥
वचन :- मायानिर्मितं पुरं मायापुरम् ॥
टीका :- तरि तें पुर कैसें होए पां : तें सांगिजतें : माया निर्मितं म्ह. निर्मिलें : तरि माया निर्मी कृतयुगीचीं मनुष्यें तैसें : म्हणें ना : तियें प्रपंचाचीं निर्मी तैसें हें नव्हे : तें पुढिलें वचनीं सांगिजैल :॥: मायापुरं म्ह. माया निर्मिलें म्हणौनि तयातें मायापुर ऐसें म्हणिजे : आणि ते आपणचि पुर जाली म्हणौनि मायापुर म्हणितलें : ॥५॥

लापिका :- तरि काइसेयाचें निर्मिलें पां? ना स्वस्वरुपें घटाकार होउनि आपणचि वोतली : ॥
वचन :- माया घटाकारें आकारे वोते ॥
टीका :- तरि तें काइसयाचें पां निर्मी : ना माया घटाकारें : घट म्ह. देह : घडिला म्हणौनि घट :।: आकारे म्ह. ते आपणचि होए : प्रपंचवत् :।: वोते म्ह. तया घटामाजि स्वरुपें : तेथ चेतनत्वें असे : ज्ञान नाहीं : यथा वोतकर्मी आधीं प्रतिमा आकारी मग रस वोती : तैसें देहाचा आकार आकारे मनोधर्मे : मग वोते स्वरुपें : तैसेंचि स्वरुप वोते ईश्वरीचें : मग तेथ विश्रांत होए : तें सज्ञान : जें भजनाते अंगीकरीत असे : तृप्ती पावत असे : एथ तीन अर्थ : घट आकारली ते जड : वोतली ते अजड : विश्रांत ते चिज्जड :॥: याचें तीन पक्ष :॥: विसरलें : पुरलें : उरलें : जड ते विसरलें : अजड ते पुरलें : चिज्जड ते उरलें : एक व्यापारीं वेचलें ते वोळगणें :॥:ओंकारबा पक्ष:॥ : देह जाली ते जड : वोतली ते अजड : तेंचि विश्रांत : व्यापारतें ते चिज्जड : हें न घडे : कां जें त्रिविधता तें मूर्तीचाचि ठाइं होआवी :॥:I: एक पुरलें : एक विसरलें : एक व्यापारीं वेचलें : पुरीची एरी पुरीचियातें न देखे भेटी होतां :I: ॥६॥

लापिका :- तरि तियेचें स्वरुप बहुधा मैळलें असे कीं? ना ते शुद्ध स्वस्वरुपें आपादमस्तक आपणचि जाली : ॥
वचन :- नख : चर्म : रोमादिकरौनि शुद्ध चैतन्यचि ते ॥
टीका :- नख, चर्म, रोमादिकरौनि म्ह. हे तिन्हीं ठाये जड : परि ते मुख्यकरौनि सप्त धातु, आपादमस्तक अवघीं मूर्ती कवळिली :॥: शुद्ध म्ह. फळीं, व्यापारीं, जीवीं मैळलें नाहीं : चैतन्यचि ते म्हणितलेया प्रपंच नव्हे : अवघी मूर्ती चैतन्याचीचि : तरि शुद्ध ते कवण : ना जे संबंधीं तिसरा अंश होउनि असे, तया समीप जें उरलें असे ते : मा ते काइ चालौनि एथ ये : म्हणे ना : ते व्यापक एथचि असे : यथा ईश्वरीचें : तरि दवडुनि कां पतित तेथ कैसें पां : ना तेंही शुद्ध चैतन्यचि : माया तें दळवाडें पालटौनि शुद्धा दळवाडेयाचें निर्मी : आकार तोचि : तया पालट नाहीं : जे कर्मवशें तें फळ रचिलें होतें तेचि अंगीकरीति : तें पुर त्यजिलेयां मागौतें पूर्वीलाचि सारिखें : ऐसी मूर्ती चैतन्याची परि नखें फेडिलेयां क्षौर केलेयां दोष नाहीं : कां जें सेवाधर्म लक्षणें गुण आति : दोष नाहीं : केवि : सेवकासि अविधी नाहीं म्हणौनि : ॥७॥

लापिका :- ऐसी आपणचि पुर जाली तरि तेथीचा अभिमान कवणासि पां? ना तो तियेसीचि : कां परमेश्वर सर्वदा निरभिमान म्हणौनि : ॥
वचन :- मायापुरी मायेसि अभिमानु ॥
टीका :- ॥मायापुरी मायेसि :॥: अभिमान म्ह. क्षेदें क्षेदु : आल्हादें आल्हादु : यथा पिंडस्थे क्षेदु आति परि अभिमान नाहीं : यथा पशुसि दुःख आति परि अभिमान नाहीं : तथा एरा : दृष्टीं सुख दुःखे तियें ईश्वरचि देति : यथा सविया ; थापा हाणिजे : ॥८॥

लापिका :- ऐसी पुराभिमानिनी सांगितली तरि वोळगणियांचा व्यापार तो अव्यक्त ब्रह्मांडवत् तियेआधीन असैल आणि काइ? ना तो सकळांचाही अवताराचि अधीन : एथचेया एकमेकावरि मनोधर्म नाहीं मा व्यापारति काइ : ॥
वचन :- ब्रह्मांडी शक्तीअधीनु शक्तीव्यापारु : मायापुरी ईश्वराधिनु शक्तीव्यापारु ॥
टीका :- ॥ब्रह्मांडी॥ मायापुर सांगतां एरें काइ प्रयोजन : तें प्रसंगे : ब्रह्मांडींचें अस्तित्व सांगितलें : आणि जीवेश्वरां साम्यालागी : सकळ देवता, वस्तु, केवळ, एवमादिक :॥: पुरीचियां एकमेकावरी मनोधर्म नाहीं मा व्यापार काइ : सकळासिही ईश्वरचि मनोधर्म : ॥९॥

लापिका :- आधीन व्यापार पां नाहीं परि अव्यक्तवत् थोरपण असैल आणि काइ? ना ते सकळ इतरशक्तीतुल्य : ॥
वचन :- मायापुरी इतरशक्तीतुल्य माया ॥
टीका :- इतरतुल्य हें व्यक्तीं : वांचौनि अव्यक्ती ते अर्धांग कीं : तेथ एरीचां ठाइं दृष्टीचि नाहीं : वस्त्रालंबनमात्रेण : ॥१०॥

लापिका :- ऐसेया पुरधार्यातें सकळ देवता वोळगति तरि तिया कैसिया पां? ना ब्रह्मांडादिकीं त्रुटि असे : एथ संपूर्णा होउनि असति : ॥
वचन :- ईश्वरीं संपूर्णा शक्ती : जीवीं अंशता शक्ती ॥
टीका :- ॥ईश्वरी संपूर्णा :॥: ईश्वरी आणि ब्रह्मांडा स्वरुपौनि येति : पिंड आणि विग्रह आरुता : पिंडीं अंश म्हणिजे : मळकर्मविशिष्टा देहियाचें ज्ञान होआवेंचि लागे : एऱ्हवीं गेला केवि जाणावा : आला तेव्हेळीं तेथौनिचि आलिया परि गेलेयाचें होआवें :॥: कलीलावस्थे शक्ती येति : ॥पंडिताचा पक्ष ऐसा॥ दृष्टी पुढां वोळगति : ॥कवीश्वराचा पक्ष ऐसा॥ मूर्तीवरि भीतरी देवता ऐसिया असति :॥: ईश्वरी संपूर्णा तरि अव्यक्तीं संपूर्णा : हें साकारीं : आणि जीवी नाहीं : तरि सृष्टीकाळीं ईश्वरें मूर्ती स्वीकरिली देवतांचिया परी तैं संपूर्णा वोळगति : तरि प्रवृत्तीसि नियम जाला :।: आणि शक्तीअधीन ब्रह्मांडी व्यापार हे न घडे : अवतारा अधीन जाला :।: मायापुर त्रिविधा परी हें न घडें : हे चौथी पर जाली :।: तरि देवांमध्ये अवतरति तेथ संपूर्णा : तेथ खालीली रिगावा नाहीं : तरि कवणें ठाइं संपूर्णा : ना कर्मभूमीचिया देवतांचां ठाइं अवतरति तेथ संपूर्णा : कर्मभूमिबाहिरी संपूर्णा नाहीं : क्षीराब्धी बीजे केलें तैं गोसावियातें वोळगति तिया खालीली राहिलिया : अर्जुनदेवाचिया पिंडीचिया राहिलिया : केवि : खालीली वरीलीतें देखणें नाहीं आणि व्यापणें नाहीं म्हणौनि :।: आणि ईश्वरें क्षीराब्धीं बीजे केलें परि आपुलाला ठाइं मूर्ती धरौनि वोळगति असति : वाट पाहाति असति : तैसिया अर्जुनदेवाचिया न पाहातिचि : तिया स्वरुपीं मिनलिया म्हणौनि अंशता :॥: आणि ईश्वरें छेद भेद अंगीकरिले परि संपूर्णा वोळगति : केवि : मूर्तीबाहिर वोळगति : मूर्तीवरी असति म्हणौनि : तेचि केवि : ना नख, चर्म, रोमादिकरौनि शुद्ध चैतन्यचि तें म्हणौनि :।: आणि आमुचां गोसावी प्रधानकुमराचें पुर (देह) स्वीकरिलें तेथ शक्ती मागिलाचि : केवि : द्वारावतिकार तेचि आमुचे गोसावीं : म्हणौनि संपूर्णा :॥:

विद्यावंतें परकाया प्रवेश केला तेथ नावेक वाट पाहाति मग व्यापका होति : तिया राहाति काइ तयांसि (तियां दासी) : पिंड उठविला तिया आणिकी : म्हणौनि अंशता :॥: देवतांचे फळिये तेथ जीवीं अंशता नाहीं : तो पिंड नव्हे म्हणौनि : षड्विकार नाहीं म्हणौनि नव्हे : तेथ मेद मांस नाहीं म्हणौनि : केवि : ब्रह्मांडी मूर्ता देवता म्हणितलिया म्हणौनि : आणि पिंडी अवघिया देवता असति : फळियाचां पिंडीं एकी बहुता एकी तोकडिया : समग्र पिंड नव्हति : भोगभूमिचां पिंडीं तोकडिया : अर्जुनदेवा क्षीराब्धीं तोकडिया :॥: हात पावो तुटलियां तोकडिया : तेथ चौरंगीची विद्या जालेयां मागौतिया येति : अर्जुनदेवां मागुतिया आलिया : भोगभूमिचे पिंड कर्मभूमि येति आणि देवता पिंडी अंश येति :॥: फळियाचां पिंडी सृष्टीपर्यंत एकी बहुता : एकी तोकडिया : तरि कर्मभूमिचिया फळियाचां ठाइं अवघिया असति : तरि पिंडहननें ब्रह्मांडहनन हें न घडे : केवि : ना तें पिंड सृष्टीपर्यंत नित्य : पिंडी देवताअंश विद्याधिकारालागी : तेथ विद्या देणें नाहीं म्हणौनि : आणि मूर्ती देवतां व्यापिली असे आणि तेथचि मेदमांसरुप होउनि असे हे अंशे :॥: आणि स्वर्गीं विद्या होए तेथ भीतरी देवता प्रकाशेना : नाहीं म्हणौनि : ब्रह्मांडस्था ये ते तयाची मूर्ती ऐसी वाखाणे : एणें न्याये जीवीं अंशता :॥: देवतांचिया मूर्ती आणि लीळाविग्रह हें निराकरिलें : पिंड शब्द मनुष्यदेहावाचक : आउसाचेनि प्रकरणें : देवताचिया मूर्ती पिंड नव्हेति : मेदमांस नाहीं म्हणौनि :॥: तिर्यंचाचां ठाइं मेदमांस असे परि विद्येयोग्य नव्हे : केवि : अंतःकरण नाहीं म्हणौनि :।: जडा स्थिति दिधली कीं : ते शिल्प म्हणौनि :॥: जीवीं मनुष्यदेहीं मेदमांस तेथ अंशता देवता असति : तेथीचेया फळियापसि यक्षणीचा फळिया पात्र : यक्षणीचेनि क्षेदाल्हादें जीवासि बहुत पुण्य पाप निफजैल : शेषशय्यापसि हे दूषण :॥:॥:

अवघेंया शक्तीसीं ईश्वर भेटे तैचि जीव उद्धरे : परि अधिकार्या : एकी शक्ती उणी कर्मभूमिची सेवटील तरि जीवासि उद्धरण नाहीं : अवरदृश्ये परदृश्ये भेटति परि उद्धरण नाहीं :।: तैसें स्थान प्रसाद शास्त्र या जडा शक्ती : वासनिकें, भिक्षुकें यां चेतना शक्ती : ऐसें अवघें भेटे तैं जीवासि उद्धरण : एकी उणी तरि जीवासि गोमटें उणें : म्हणौनि बहुतें भिक्षुकें, बहुतें वासनिकें, बहुतें निरुपणें, बहुत प्रसाद ऐसें भेटे तैं सर्व शक्तीसि युक्त परमेश्वर भेटला होए : एसणें गोमटें फळें : जैसा चालता बोलता भेटला एसणे गोमटें फळें : ॥११॥

लापिका :- ऐसेया मायापुरातें अंगीकरुनि काइ करीति पां? ना कर्मक्षिती वर्तमाना मानवदेह संपूर्णा जीवासि ज्ञानावधी स्वसंबंध देति : ॥
वचन :- ऐसेया मायापुराते स्वीकरौनि कर्मभूमिस्थितां मनुष्यदेहयुक्ता जीवा ज्ञानपर्यंत परमेश्वरु आपुला संबंधु देति ॥
टीका :- ऐसेया म्ह. निर्वाळा सांगिजताये : मागिले ठाउनि : सृष्टीमध्यें अवतार : मनुष्यवेष : मायावेष : मायापुर : देहधर्म : जीवधर्म : देवतांचे धर्म : मायेचे धर्म : जडाचेतना शक्ती : दवडौनि कां पतित स्वीकरिलें : जें पाइका न ल्हाये ऐसें होतें तया एसणें सामथ्र्य आलें : जें च्यारि दानें दे : सकळ देवता वोळगति : सृष्टीचेया जीवातें उद्धरी : ऐसें परिपूर्ण जालें म्हणौनि पुर शब्द :॥: स्वीकरौनि म्ह. तयाते अंगीकरुनि :॥: परमेश्वर म्ह. उभयदृश्या : भलते करुं सके : जीवाचा ईश्वर करुं सके :॥: कर्मभूमि संख्या सांगितली तें :॥: स्थितां म्ह. तेथ राहिले असति तयां : तरि भलते जीव : म्हणें ना : जें मनुष्यदेहातें पातले तयां : तरि जेतुलें मनुष्यदेहातें पातले तयां : म्हणें ना : जें युक्त तयांसि : ना अवघेया आंगीं पुरतें : सर्वावयवीं संपूर्ण : चातुष्ट्य युक्त : हे असे तें मनुष्य : एणेंकरौनि वेडे पिसे निराकरिलें :॥: जीवा म्ह. जीवासीचि : संलग्न जीव जालें असे तें ना :I।: ज्ञानपर्यंत म्ह. कृपा, साह्य, गिवसणें ; अधिकार घडवणें ; ज्ञान देणें : एतुलें ज्ञानपर्यंत करीति : मग तयाचा कड पाहाति : तैसेंचि जीवें आपुला कड केलेयां तयाचा कड न दिसे तरि अव्हेरावें :।I:॥:॥: ज्ञान देति कारणें ईश्वरीची चौदा आणि जीवाचे सात अधिकार ऐसें एकवीस वर्तति तैं देति :॥: तियेचि कवणें :॥: सृष्टी, कर्मभूमि, मायापुर, मनुष्य अवतार, देहधर्म, जीवधर्म, मायेचे धर्म, देवतांचे धर्म, गुण, धर्म, विज्ञान, अवरशक्ती, परशक्ती, प्रवृत्ती : हें ईश्वरीची चौदा :॥: तरि जीवाचे सात अधिकार तें कवण : ना कर्मभूमी, मनुष्यत्व, युक्तत्व, कर्माधिकार, देहाधिकार, मुळाधिकार, ज्ञानाधिकार : आणि घासाचिये ठाउनि नवा दृष्टांताचे अनधिकार न व्हावे : तेव्हेळीं जीवासि ज्ञान देति :॥:॥:I।: कर्मभूमी कां सांगितली : आणिके ठाइं कर्म न निफजे हाचि विशेष : मनुष्यत्व कां सांगितले : ज्ञानालागी : पिंडस्था कां : विद्येलागी : पशुसि कर्म न निफजे हे कृपा :I।: एकासि मनुष्यदेह आपण देति : एकासि अधिकार घडवीति : आभारैलेपणें भजन स्वीकरीति : भोगभूमीचा कर्मभूमी आणीति :।I: गोसावीं जे देह दिधलें तेथ पापाचा लेप न लगे : पुण्याचा लागे : सुख दुःखें भोगी तेथ पापाचा अंश तो आणिके जन्मींचा नासे : पुण्याचा रासी वाढे :।I: ॥१२॥

लापिका :- तरि काइ ज्ञानचि देति? ना उत्तमाधिकार्यासि प्रेम देति : ॥
वचन :- एकां प्रेम संचरीति ॥
टीका :- एका म्ह. जे पुरभजनाचे : तयांची हे व्यवस्था : ज्ञानियाचिया ऐसीचि : परि ते आन जाति : प्रेम म्ह. वियोगीं नुरे :॥: प्रेमाचा संचार तो दृष्टीद्वारकचि : उभयाचिये : आंधळेयां पाठिमोरेयां न संचरे : कां जें लोकीचीं प्रेमें तियें दृष्टीचि कीं : ॥१३॥

लापिका :- तरि काइ उत्तमा साधनासि हेचि नाम? ना प्रेमाचिये उत्तरावस्थेसि भक्ती म्हणितले : ॥
वचन :- प्रेम बोलिजे : भक्ती बोलिजे ॥
टीका :- प्रेम तेंचि भक्ती : तरि नामभेद कां : ना प्रेमाची उत्तरावस्था तें भक्ती : यथा बोधाची उत्तरावस्था ते ज्ञान : नसुधेनि बोधें बोधवंत तोचि शास्त्रें दृढावलेयां ज्ञानिया म्हणिजे : कोमळ तो बोधवंत : जरठ तो ज्ञानिया : तैसे कोमळ तो प्रेमिया : जरठ तो भक्त : म्हणौनि दोहीचें एकचि वाक्य : प्रितिची उत्तरावस्था तें प्रेम : प्रेमाची उत्तरावस्था ते भक्ती : ॥१४॥

लापिका :- तरि ऐसेनि प्रेमें काइ होए पां? ना संचारमात्रेंचि विकारादि दोषत्रय उच्छेद : ॥
वचन :- प्रेमसंचारमात्रे विकारु : विकल्पु : स्वभावो इये तीनि उच्छेदौनि जाति ॥
टीका :- ॥उच्छेद॥ प्रेमाचां संचारीं मनोद्वंद्वें नाहीं : कां पां : विकार उच्छेदला म्हणौनि : विकारे मनोद्वंद्वें : विकल्पें अन्यथाज्ञान : स्वभावें देहद्वंद्वें : तिहींकरुनिं तिन्हीं जाति :।I: संचरत संचरत उच्छेद :।I: मात्र शब्द अल्पकाळावाचक : प्रेमें स्वभावो : ज्ञानें विकल्प : विकार वैराग्यें उच्छेद :॥: स्वभावो दिसत असे तरि उच्छेद कां म्हणितला : ना कार्य नाहीं म्हणौनि :।I: ॥१५॥

लापिका :- तरि काइ भक्ताचा हाचि विशेष? ना ईश्वरींचेनि आत्मत्वें सकळही सन्निधानीं चरितार्थ होए : ॥
वचन :- भक्ताचे सकळही सन्निधानी चरितार्थ होए ॥
टीका :- सकळही चरितार्थ म्ह. शाब्द चरितार्थ ; चहुं ज्ञानींचीं विघ्नें ; आचार : एतुलें बैसलेयाचि चरितार्थ :।: सन्निधान म्ह. सम्यक्प्रकारें निधान म्हणौनि सन्निधान :I।: दुसरें : चरितार्थ म्हणितलेयां आचरणेंकरौनि जो अर्थ तें चरितार्थ : जेतुलें ज्ञानियाचें करण तेतुलें करितां होए : श्रीमूर्तीचे : नामाचें :।: विशेषज्ञानीं कृपा दायेतेचें दुःख नाहीं : प्राप्तीचे नाहीं : ॥१६॥

लापिका :- तरि भक्तासि श्रीमूर्तीधार्या परमेश्वरावरी बुद्धी कैसी पां? ना हाचि माझें साध्य आणि साधन ; जे पाविजे तें तथा पावविता ऐसी प्रतिती, निश्चयो : ॥
वचन :- भक्तासि श्रीमूर्तीवरी साध्य साधन ऐसी प्रतिती : प्राप्य प्रापक बोधु ॥
टीका :- भक्ता म्हणितलेया ज्ञानिया निराकरिला :॥: श्रीमूर्ती म्ह. श्री जिया अलंकरिली : मूर्ती म्ह. मवली :॥: तरि आधीं साध्यचि कां म्हणितले : ना प्रेमाचा संचारी थोर बुद्धी बैसली : साध्य ऐसेंचि उमटलें तरि तेंचि आधीं सांगितलें : साध्य म्ह. स्वरुप : साधन म्ह. साध्यातें देतें जें : प्रतिती म्ह. प्रत्यया ऐसें आलें ; जें दोन्हीं ठाये मूर्तीचि होए : साध्य एक वेगळें नाहीं :॥: प्राप्य म्ह. जें पाविजे तें : प्रापक म्ह. तयातें पावविता : बोध म्ह. हें दोन्हीं ठाये मूर्तीचि होए ऐसा निश्चयो पातला :।: तरि प्राप्या साध्या कैसा भेद : ना स्वरुपातें पावणें तें साध्य : तें पावौनि तेथीचा अनुभव लाहिजे तें प्राप्य : यथा देवताचां ठाइं प्राप्तीफळ :॥: साधना प्रापकां भेद : यथा पिंडस्था साधन होए परि साध्य पावविती नव्हे : तयाची व्यावृत्ती : मूर्ती दोन्हीं आपणचि :॥: प्रतिती बोधां भेद : ज्ञानासवे हा ईश्वर ऐसें प्रत्यया आलें तें प्रतिती : तेचि शास्त्रें दृढावे, जीवीं निश्चयो पावे, बैसोनि ठाके तो बोध : एकी पूर्वावस्था, एकी उत्तरावस्था :॥:०:॥:I: भक्ताची प्रतिती जैसी विद्यावंता : पिंडस्थ साधन : वरील स्थान मूर्ती प्राप्य : सुख साध्या नेति तेचि :।I: ॥१७॥

लापिका :- तरि साधनरुपा ईश्वराचां ठाइं साध्य प्राप्य बुद्धी कां पां? ना जे साध्य तेचि साधनरुप व्यक्त जालें आणि करचरणवंत ब्रह्म हें ऐसा निश्चयो म्हणौनि : ॥
वचन :- साध्यसाधनाकारे आकारले करचरणवंत ब्रह्म हें ऐसी भक्तासि प्रतिती ॥
टीका :- साध्यसाधनाकारें म्ह. जें साध्य तेंचि साधनरुप व्यक्त जालें :।: करचरणवंत म्ह. तयासिचि हस्तपादादिक निगालें : सावयव ब्रह्म हें : ऐसीचि प्रतिती मूर्तीवरी भक्तासि : ॥१८॥

लापिका :- तरि काइ हाचि विशेष? ना साधन संचारें जें आलें आत्मत्व तयास्तव वियोगीं उरों न सके : ॥
वचन :- भक्त वियोगी नुरे ॥
टीका :- वियोगी नुरें तो कोमळ : म्हणियां धाडिजे तेथ देह धाडी तें नव्हे : हे वाक्य एकदेशीं : यदा कदा बीजे करीति जैं : ॥१९॥

लापिका :- हो कां : ईश्वरें वियोग दिधलेयां देह त्यजी हाचि विशेष? ना ईश्वरीचिया प्रिती तो क्षणही भर पैऱ्हां जावों न सके : तथा ईश्वरही त{त्प्रती तयातें केधवांही आपणया वेगळें न करीति : ज्ञानियाची व्यावृत्ती : ॥
वचन :- भक्ता नित्य संबंधु ॥
टीका :- नित्य तो भक्ताचिया कडौनि : आपुलिया आवडीं कां प्रयोजने परि जाणें आतिचिना : ज्ञानियाची व्यावृत्ती केली : बारा वरुषें अज्ञानदशा तवं वेवधान : तें ईश्वरीचिया कडौनि : तें तो जाणेंही ना : ॥२०॥

लापिका :- भक्ताची व्यवस्था पां ऐसी तरि भूतभजनियासि ज्ञान देति ते कैसें पां? ना तें वैखरीद्वारें परा व्यापारें सृष्टीमध्यें परमेश्वर बोलिजे तो हा ऐसा निश्चयो करीति : ॥
वचन :- एका जीवाप्रति परागिराकरौनि परमेश्वरु आपुले अस्तित्व साधीति ॥
टीका :- परा म्ह. परा : गिरा म्ह. वैखरी : दोहींकरौनि : आपुलें म्ह. स्वरुपींचें : अस्तित्व म्ह. शाब्देंचि : साधीति म्ह. परमेश्वर एक सृष्टीमध्ये बोलिजे तो इये मूर्तीआंत सिद्ध केला : यथा निक्षेपाचा संदेह होता तो एथ असे ऐसें केलें : जीव मागां होता तो पुढां केला : रुप मागां, साइली पुढां : म्हणौनिचि परमेश्वर : ॥२१॥

लापिका :- ऐसा परमेश्वरें बोध दिधलेयां मग जीव काइ करीति पां? ना एक अनुसरेति, एक ननुसरेति : हे जीवाची स्वातंत्री सांगितली : ॥
वचन :- मग एक अनुसरेति : एक ननुसरेति ॥
टीका :- एक अनुसरेति एक ननुसरेति म्ह. हे जीवाची स्वातंत्री सांगितली : जेतुलियां ज्ञान तेतुलें अनुसरेति ऐसें नाहीं : अनुसरेति ते मनुष्याणां सहस्रेषु : ईश्वर आपुला कड करीति : वरिचील कड जीवाचा : अनुसरे ननुसरे दोन्ही आति : ॥२२॥

लापिका :- ननुसरेति तयांसि कवण गति पां? ना तें पतित अनार्थिये तयासि सृष्टी निर्फळ होए : तरि ज्ञानें चैतन्यफळ, कणवा, तिन्हीं दानें, उपायें देहविद्यादि सकळ फळें पावत असे कीं? ना तो सृष्टीअंतरवृत्य निस्तरा : परि तियें च्युतिमंते : कैवल्यदान तें आतिचि ना : यातें शून्य बोलिजे : ॥
वचन :- ननुसरेति ते प्रमादिये बोलिजेति : तेया सृष्टी शून्य होए ॥
टीका :- ननुसरेति तें प्रमादिये म्ह. पतित, अनर्थिये, ईश्वर दुऱ्हावे : या परतें पतन, अनर्थचि नाहीं :॥: सृष्टी शून्य म्ह. तया सृष्टी निर्फळ : वांझास जाए : ईश्वरेंसिं सोयरिकेचा सांदा तुटे : तिये सृष्टी पुनरुपि ज्ञान नेदीति : तिन्हीं दानें, दुःखापासौनि सोडवीति ; हे ईश्वरीची कणव होए, माउलीपण होए : परि उद्धरण हेतूचा सांदा तो नाहीं यातें शून्य बोलिजे : वासनिकाचा लेपाधिकार मोडला नाहीं : अनुसरलेया वांचौनि : ननुसरे तियें जातिचें वोखटें आति : ॥२३॥

लापिका :- तरि अनुसरलेयासि कवण गति पां? ना प्रस्तुती कर्माची त्रिविधता होए : ॥
वचन :- अनुसरलेयाचे आगंतुक कुंठे : अनारब्ध सुंके : प्रारब्धा भोगे क्षयो ॥
टीका :- कर्माचिये व्यवस्थे आधीं आगांतुकें : कां जें झरा आधीं बुजावा मग उरलें पाणी सांडावें : एसणें ज्ञान, सन्निधान, अनुसरण, दास्य एणें थोरचि निफजैल म्हणौनि आधीं अधिकार मोडिला : तेणें अनारब्धा पाक हों पारुखला : जैसें धान्य उगवतां पाउस न पडेचि तेणें बीं उगवेचि ना : एक उकांणियां पोटरियां आलें तें सुकलें : पक्व तें भोग्य :।I: अनागत अनारब्ध तें अनुसरण-कार्य : प्रारब्ध तें ऐसेंचि निरुपण : पूर्विल व्यवस्था : प्रस्तुत दोन्हीं आति : यथा जुंपीवत् : ॥२४॥

लापिका :- प्रारब्धां भोगें क्षयो म्हणितले तरि जैसा तैसा भोग होईल कीं? ना शरणांगत लक्ष कवडे न्यायें तोकडाचि : तोही ईश्वरदास्यें : कर्मसवा नाहीं : ॥
वचन :- अनुसरलेयासि कर्मसवा भोगु नाहीं : दैवसवा भोगु ॥
टीका :- यातवं जैसा भोग तैसा आपजैल म्हणौनि दैवसवा एणें निराकरिलें(ला) : यातवं भोग तोकडा जाला : ॥२५॥

लापिका :- तरि अपरोक्षीं कर्मानुसारें देव तिर्यंचादि रुपें धरौनि क्षाळील कीं? ना अनुसरलेयासि दोन्हीं होणें नाहीं : म्हणौनि मनुष्यदेहीं बहुरुपता : एरी दो ठाइं कामनाचि : ॥
वचन :- अनुसरलेयासि देवत्व : तिर्यंचत्व होणे नाहीं ॥
टीका :- परि स्वर्ग नरक न चुकति तें वरिलें निराकरिलें : एवं भोग तो मनुष्यदेहीं : तरि पदार्थ अमृतादिक आणावें : मा हें जरि आणावें तरि देवता, परिवार समग्रावांचौनि भोग नाहीं : म्हणौनि हें दूषण : ॥२६॥

लापिका :- तरि ज्ञानियासि श्रीमूर्तीधार्यावरी कैसी बुद्धी पां? ना हा माझें साधन तथा साध्यातें पावविता ऐसी प्रतिती, निश्चयो : ॥
वचन :- ज्ञानियासि श्रीमूर्तीवरी साधन प्रतिती : प्रापकु बोधु ॥
टीका :- ज्ञानियासि श्रीमूर्तीवरी प्रतिती ते भक्ताचियाचि सारिखी : मग प्रापक हे बुझविला :॥: दुसरा पक्ष :॥: हा ईश्वर ऐसें जाणविलें पूर्वीचि : मग साध्य साधन दोन्हीं शास्त्रें बुझविलें :॥: दुसरा अर्थ :॥: परा म्ह. परा : गिरा म्ह. वैखरी : कां जें दोहीं मिळौनि होत असे कीं : दोहीं अर्थीं विचार असे : उपपत्ति असति बहुती : तयाचा कड जालेयां ईश्वर दो घडिया अंगीकरीति : मग कर्मव्यवस्था : अनुकुळ प्रतिकुळत्व : जीवासि जेव्हेंळीं ईश्वरप्रतिती तेव्हेंळीं चैतन्याची जे पूर्विल तेचि उमटे : हा ईश्वर ऐसी प्रतिती जालेयांचि प्रेतदेह नासलें : कां अधमबुद्धी गेली : यथा जोगी : मग हा म्हणे तें उत्तम : एर अवघीं कल्पना गेली : जेणें आरंभे : ॥२७॥

लापिका :- हो कां : साध्यातें पावविता म्हणितले तरि साध्य कवण पां? ना मोक्ष तथा कैवल्य : दोहींचाही दाता हाचि : ज्ञानस्वरुपता ऐसी स्वपर जें जैसें असे तें तैंसे जाणिजे म्हणौनि : ॥
वचन :- कैवल्यदाता हा : मोक्षदाता हा : ऐसी ज्ञानियासि प्रतिती ॥
टीका :- ॥मोक्षदाता॥ मोक्ष तो बंध त्रुटीसि वाचक होआवा : कां तरि अन्यत्र प्राप्तीसि असे तैसाचि होआवा :॥: मोक्ष म्ह. संसारनिवृत्ति आणि ब्रह्मस्वरुपीं असणें : कैवल्य म्ह. ईश्वरीची प्राप्ती : मोक्षातेंही कैवल्य म्हणिजे : तें अपेक्षिक : यथा अनुसरला भक्त : हें साध्यपर : साधनीं तरि प्रेमातें कैवल्य म्हणिजे : ब्रह्मविद्येतेंही म्हणिजे : दाता म्ह. दोहींचा दानी : ॥२८॥

लापिका :- ऐसा निश्चयो जालेयां ज्ञानियां काइ करी पां? ना सुमती तो अनुसरौनि सन्निधान करी : तत्कृतार्थता एणें योग्य जालेयां परमेश्वर पदार्थ चतुष्टयाचा साक्षात्कार करीति : ॥
वचन :- अनुसरौनि सन्निधान करी : या परी योग्य होए : तयासि परमेश्वरु सत्ताज्ञान दान करीति ॥
टीका :- ॥अनुसरौनि॥ अंतरें अधीनता बाह्य सेवादास्य तें अनुसरण : तरि मागांचि अनुसरला कीं : ना ननुसरती तयांची व्यवस्था सांगितली तियेचि पांतीं अनुसरण-कार्य सांगितलें : परि तें नाहीं :॥: सन्निधान करी म्ह. जवळीकें सहवासावाचक : कां जें म्हाइंभटांतें म्हणितले : हा सन्निधान पातला असे : तुम्हीं पावावें असा : ज्ञान, अनुसरण, संग, जवळीक, सेवा, दास्य, भजन, निरुपण एतुलेंयातें सन्निधान पातला बोलिजे :॥: या परी म्ह. मुळाधिकार : योग्यता दुःखीं, कष्टीं : योग्य म्ह. पात्र : तयासि म्ह. जो शाब्दें निर्वाळला : एरा नाहीं :॥: परमेश्वर म्ह. जो उद्धरिता : सत्ताज्ञान म्ह. शाब्दीं सांगितलें जें च्यारि ठाये तया अस्तित्वाची सत्ता : करीति म्ह. दर्शन : यातें सत्ताज्ञान बोलिजे : दान म्ह. सत्ता परित्यागें देति : ॥२९॥

लापिका :- तरि तो साक्षात्कार कैसा करीति पां? ना उचिते समयीं सकळ मळ अपसारुनि जीवस्वरुपीं सामथ्र्य संचरीति : ॥
वचन :- कव्हणी एकी अवसरीं सकळही मळांचे परिसारण करुनि परमेश्वरु जीवीं आपुली सत्ताशक्ती संचरीति : मग मागौते मळ घालीति ॥
टीका :- कव्हणी एकी अवसरीं म्ह. पूजा अवसरीं : तेचि पूजा व्यतिक्रमें डोमेग्रामीं करविली : बेलोपुरी करविली : तियेसारिखी कीं अनारिसी हें नेणिजे : तीं प्रकारीं करवीति : थोरा नासाचेया हेतू : थोर योग्यता जोडावेया : साधन चरित करावेया : सामान्यज्ञान विशेषज्ञान तेंहीं पूजावसरींचि देति :॥: सकळ मळ म्ह. यक्षणीचिये ठाउनि विश्ववेऱ्हीं : परिसारण म्ह. परते सारीति : दृष्टीकरौनि : यथा बाबुळी : आद्यमळ तो जीवेसीं युक्त असे : तथा अविद्याही : दोन्हीं कृपेचिया अणिया भेदिती :॥: मळ घालीति म्ह. दृष्टी हालवीति आणि मिळति : ॥३०॥

लापिका :- तरि पदार्थ चातुष्टय देखे हाचि विशेष? ना फळसूचक देखे आणि सर्वदा अनश्वर (अनीश्वर) : ॥
वचन :- सत्ता बोलिजे : सूचा बोलिजे : अव्यभिचार्य ज्ञान बोलिजे ॥
टीका :- सत्ता म्ह. अस्तित्वाचें दर्शन : हें याचि शब्दांत असे : पुढिलें वचनें याचिया विशेषातें साधितें असति : ॥नागाइसाचा पक्ष॥ प्राप्तीची सुचा स्फुरे : जीवीं सुचा स्फुरे :॥: अव्यभिचारियें एरें ज्ञानें देवतांची : कोमळ : दिवा ; सूर्य :॥: साधन काइसेनिही न वचे तो अव्यभिचार :I।: मग प्रेमाचा उपावो न प्रभवे तो अव्यभिचार : बाइसां बोध आति : आणिक बलवत्तर देणें नाहीं : फळ होएचि ते सुचा : एक साधनपर : एक साध्यपर :।I: ॥३१॥

लापिका :- तरि काइ हाचि विशेष? ना दुरादर्शिन्यायें पदार्थ चातुष्टय समग्र देखे : ॥
वचन :- सत्ता ते विदित ॥
टीका :- ॥सत्ता ते विदित॥ आम्हां समुद्राची सत्ता जाली परि पैल सेवट अनुमानेना, ऐसें आपजैल तें विदित म्हणौनि निराकरिलें : मग समग्राचें दर्शन आपजलें : ॥३२॥

लापिका :- तरि काइ हाचि विशेष? ना जया पदार्थाचें जें लक्षण तें यथावत् देखे : अन्यथात्व एतुलेंही नाहीं : ॥
वचन :- सत्ताज्ञान तेंचि अपरोक्षज्ञान ॥
टीका :- सत्ताज्ञान तेचि अपरोक्षज्ञान : परि व्यवधानें दर्शन अन्यथात्व असे तें अपरोक्ष शब्दें सांगितलें : अपरोक्ष म्ह. तेथीचियाचि विशेषातें साधित असे : ॥३३॥

लापिका :- तरि काइ हाचि विशेष? ना समग्र कर्मे तथा तत्क्षाळणोपाय जाणे : आणि विघ्नागम तथा तदोपशमोपाय जाणे : ॥
वचन :- सत्ताज्ञानी अशेष कर्म जाणे : कर्मक्षाळणी उपावो जाणे : येते विघ्न जाणे : विघ्नोपशमु जाणे ॥
टीका :- सृष्ट्यांतरीचेनि पोळे तो अपरोक्षज्ञानिया : कां कर्म जाणे म्हणौनि :॥: अशेष कर्म म्ह. अनंता सृष्टीचीं विचित्रें : जाणणें हें अनुभवावाचक : अनुभव तो प्रत्यक्ष देखिलेयांचि : कां जें अनुभवें जाणाल म्हणितलें : तरि देखेचि म्हणावें कीं : ना (मा) लेप नाहीं ऐसीं एकें असति तेथ काइ देखावें : जया असे तियें देखें : म्हणौनि दोन्हीं साधावेया जाणे हेंचि उचित :॥: उपाये जाणे म्ह. तें जया परी नासति तो प्रकार जाणे : कर्मांत मळही लाभति : कां तेंही केलेचि कीं : परि तयांचा नाश एथ नाहीं : तो सामान्यीं : परि उपावो एथचि जाणितला :॥: विघ्न म्ह. विशेषें हाणति : हाणति म्ह. साधना विघात करीति : पुरुषातें टाळीति : ते गुणचि (गुणेचि) की : परि धर्मा विघातक : येतें म्ह. येईल ऐसें भविष्य जाणे : ऐसें जाणे तरि व्यवधाना कां जाए : ना अभिलाषें : सन्निधानी न पवे : व्यवधानीं पावें : उपशम म्ह. तें जेणें शमे तो उपावो जाणे : ॥३४॥

लापिका :- कर्मक्षाळणोपावो जाणे म्हणितले तरि कैसें कवणे ठाइं क्षाळी पां? ना सन्निधानी, असन्निधानी : ॥
वचन :- सन्निधानी क्षाळी कां असन्निधानी क्षाळी ॥
टीका :- सन्निधान म्ह. मूर्तीजवळी हो अथवा वचनाचें तेंही सन्निधानचि : सन्निधान तें ऐसेयावाचक : अनावसरीं अवसर ; प्रवृत्तीचा अभिनिवेश ; क्रियेचा अंगीकार ; सेवा, दास्य, व्यापार तें सन्निधान : हें नाहीं तें असन्निधान : वांचौनि शाब्दीं मूर्ती आणिके ठाइं आणि तो आणिके ठाइं ऐसें नव्हे : हें तया नाहीं : तरि असन्निधान तें केव्हेळीं : ना पहुड असे कां कव्हणाचिया खंति औदास्य असे, कां उद्धरण व्यापारें आणिकां ठायां बीजे करीति : तवं तो आपुलिया कामा प्रवर्ते : दों ठाइं क्षाळण सांगितले : ज्ञानियासि कर्में क्षाळिता आपुलिये स्वरुपींचे फिटावें हें लक्ष : देवतांचे ॠण हे नांतर्येक : जैसें सात्विका ॠण फेडावें आपुलीं हाडे मोकळीं करावीं :।I: ईश्वरासि देवतांचे लक्ष : कां तेही सेवकें : आणि तो नीतिमंत कीं : एकाची अवज्ञा न लेखणें हेचि उचित : यथा मासिया : ॥३५॥

लापिका :- तरि दोहीं ठायांचा विशेषाविशेष कवण पां? ना सन्निधानी तोकडेनि बहुत : एरी ठाइं जेतुलें तेतुलेंचि : ॥
वचन :- सन्निधानी तोकडेनी बहुत क्षाळी : असन्निधानी जे जेतुलेनि क्षाळावे तें तेतुलेनि क्षाळी ॥
टीका :- तोकडेनि बहुत म्ह. काळें आणि व्यापारें : हें सन्निधान शाब्दीं (शब्दे): एक सेवट मूर्तीनिष्ट व्यापार करी : बोणें : आरोगणा : मर्दना : एक साधिकरण : स्वनिष्टीं आपुली झोळी : मग उत्तरोत्तर दोहीं उणें जाणावें :॥:॥: जें जेतुलेनि म्ह. जियें कर्मीचें जैसें अनुकरण तें समग्र करी : एक प्रतिक्षणीं करितां काळही बहुत तरि कैसें पां क्षाळी : ना मनुष्यदेहीचीं तियें तेणेंचि : या खालौतीं वरौतीं तिये तेथ कामनाचि : कां जें अनुसरलेयासि दोनि ठाये होणें नाहीं म्हणितले : मनुष्यदेहीं बहुरुपता : सिधे नाहीं तेथ आपणचि रची : तयांतु संचरे : यथा ग्रहोवत् : तरि काइ एकें वेळें : ना अनुक्रमें : एक करुनि एक : कां जें अपरोक्षज्ञानियासि बहुरुपता म्हणितलीं :॥:I।: अपरोक्षविद्यावंत देह रचू सके : केवि : गरिमा आणिमा सांगितलिया म्हणौनि : अपरोक्षज्ञानासवें जीवातें, ईश्वरातें, देवतातें सामान्यत्वें देखे : विशेषज्ञानीं विशेषत्वें देखे : देवतातें कवणें ठाइं सांगितले असे : ना अपरोक्षा होति म्हणितलें : जीवाचें कैं सांगितले : तथा प्रमाण सांगितला : माया कृपा वेगळिया सांगितलिया : तयाचि तवं सकळ ब्रह्मांडीचिया जीवा :।I:०:I।: बहुरुपता तें अपरोक्षीं : कामना तें विशेषीं : कां जें सनहित तवं तवं चाड : यथा भोजन : यथा संबंधी विकार : भक्त आणि कामना कैसी : तरि तयासि अविद्या छेदितां, प्राप्ती करितां काइ दुःख होए (होत असे) : तरि साली गोड (गोडी) कैसी : भक्ता आणि ज्ञानियाचें स्वरुप मिसळलें : परि प्राप्ती अनारिसी : कां जें दोही वाट फांकली : तेथौनि वाक्येंची आनें : यांत अर्थ असे :।I: ॥३६॥

लापिका :- विघ्नोपशम जाणे म्हणितलें तरि विघ्न कवणाचें पां? ना शाब्दीं दोनि सृष्टी विघ्नरुपा होतिया तिया एथ अधीना : दैवसृष्टी विघ्नरुपा : ॥
वचन :- सत्ताज्ञानी विकारसृष्टी कर्मसृष्टी अधीना होति : दैव विघ्न ॥
टीका :- विकारसृष्टी म्ह. प्रतिकुळा : कां जें विकरौनि पुरुषांसि विघ्न करीति : कर्मसृष्टी म्ह. अनुकुळा : कां जें पुरुषासि विधीचा क्रम करवीति : या इंद्रादिकांखालौतिया दो थोवा जालिया : एकीचि स्थानीं द्विधा : कैसिया : ना स्त्रिया तेतुलिया प्रतिकुळा : पुरुष तेतुले अनुकुळ : हें अनुसरण-कार्याची पूर्वव्यवस्था : परि ईश्वरें कणवेतवं अनुकुळा सबळा केलिया : कां जें कोमळज्ञानी पुरुष निर्बळ : दोषांचें उलंबन बहुत : आणि प्रतिकुळा सबळा तरि याचें काइ उरें : आधीना म्ह. अनुषंगिका होति : याचें समर्थपण देखौनि रंका होति :॥: दैव म्ह. ब्रह्मादिकावरौतें चैतन्यपर्यंत देवता विघ्नरुपा होति : शाब्दीं अपाड : एथ सरिसा पाड जाला : एथ अनुकुळत्व नाहीं : ॥३७॥

लापिका :- तरि दैवसृष्टी विघ्न कैसें करीति पां? ना ब्रह्मादिक जय जय (जय जीये) शब्दीं प्रार्थौंनि आपुलिया फळाप्रति नेयावेया येति : परि तो ज्ञानिया तयातें इच्छीना : ॥
वचन :- ब्रह्मादिक आपुलेया पदा नेयावया येति परि तो तयांची वासही न पाहे ॥
टीका :- ब्रह्मादिक म्हणितलेया शेषशय्याही लाभे : कां जें चहुतें महाणव म्हणितलें : प्रतिती सारिखीचि :॥: पद म्ह. फळावाचक, स्थानावाचक : नेवों येति म्ह. रंकपणें आदरेसीं :॥: वास न पाहे म्ह. येउलीही इच्छा न धरी : यथा सेनपुंजियेसि चक्रवर्तीची वासना : परि तो वासही न पाहे : यथा मळवत् : ॥३८॥

लापिका :- तथा भैरव आपुलेनि प्राक्रमें, प्रभावें, बलात्कारें नेवों पाहाति : परि तो अचल सिध्यंगिकारें अति घनवट होए तयास्तव काहींचि नव्हवे : ॥

लापिका :- या परी पात्र (योग्य) जालेयां पुरुषासि परमेश्वर काइ देति पां? ना विशेषज्ञान देति : तेणें समग्र यथातथ्यें प्रकाशलें : काहींचि उरी नाहीं अनुभवावांचौनि : ॥

टीका :- या परीं म्ह. अनुक्रमें : मळकर्मे क्षाळुनि : योग्य म्ह. अधिकारिया, पात्र : तयासि म्ह. जो पात्र जाला असे : परमेश्वर म्ह. जो उद्धरिता, रक्षिता, तारिता :॥: विशेष म्ह. तयापरतें नाहीं : अवघेंया ज्ञानाचा विशेष : जीव देवता परमेश्वर तिन्हीं स्वरुपें सविशेषें देखे : अपरोक्षी झावळें देखत होता तें सामान्यीं नावेक व्यक्त देखे : विशेषीं सविशेष देखिलें : केवि : दुरादर्शियाचेनि दृष्टांते : म्हणौनिचि ज्ञान :॥: दान म्ह. उपकारनिरपेक्ष तें : करीति म्ह. सिधी (सिद्धी/सिद्ध) :॥: तरि काइ च्यारि वेगळालीं : म्हणें ना : विद्या एकीचि जे अपरोक्षीं दिधली : तियेंचेचि संपूर्णत्व होत जाए म्हणौनि करीति म्हणितलें : आणि पूर्णत्व तवं तवं देखणें म्हणौनि तीहीं ठाइं ज्ञान शब्द : विशेषज्ञानीं मळ सारावे न लगति : कां तें मागांचि नासिलें : ॥४७॥

लापिका :- हें ज्ञान दिधलेयां काइ होए पां? ना दैवसृष्टी अधीना होति तिये वेगळा : आणि सकळाही शक्ती अधीना अपरोक्षा होति : ॥
वचन :- विशेषज्ञानी दैवसृष्टी वेगळा होए : सकळाही शक्ती अधीना होति : अपरोक्षा होति ॥
टीका :- दैवसृष्टी वेगळा म्ह. मागिलाचि परी : कां जेचि विघ्न तेचि अधीन, तयासींचि वेगळा होए म्हणितलें :॥: शक्ती अधीना म्हणिजेति ब्रह्मांडीचिया सकळा देवता : अनुकुळ मनोधर्म : यथा भक्त एकमेका : ईश्वरा खालौता : आणि जडाही वशवर्तिनी होति : परि तो नेणैल म्हणौनि अपरोक्ष शब्द जाला : यथा दुर्योधनवत् :॥: अपरोक्षा म्ह. विशेष शक्ती देखे : अनुकुळा कां : ईश्वरा समान जाला : ॥रामेश्वरबास॥ विशेषज्ञानीं अधीना अपरोक्षा म्हणिजेति येचि ब्रह्मांडीचिया शक्ती : कां जें मागिला प्रकरणातवं : मग दान तें सकळा ब्रह्मांडीचिया : जडाही आदिकरौनि :॥: विशेषज्ञानियातें देवता देखे : बाहिरील प्रपंच असेतवं देखे : मग न देखे : ॥४८॥

लापिका :- सकळा शक्ती अधीना म्हणितलियां तरि माया काइ पां? ना तियेचा कोप न वचे : परि काहीं करुं न सके : कां चक्रवर्तीवत् समर्थ जाला म्हणौनि : ॥
वचन :- माया कोपौनि उदासिन होए ॥
टीका :- कोपौनि उदास यथा माता पुत्रातें अन्येकीं परी शिकवितांही पुत्र शब्द न करी ; तेव्हेळीं तयासि भलतें हो म्हणौनि उदास होए : तैसें माया विघ्न करुनि पाहिलें : परि तियेचिया मोहरा नव्हेचि : तेव्हेंळी तिया आस सांडिली यातें उदास बोलिजे :॥:I।: माया म्ह. ईश्वर जयाची कृपा करीति तयातें खावो रिगें : अव्हेरीति तयासि आपुले पद दे : ईश्वरीचा अंश असे म्हणौनि :।I: ॥४९॥

लापिका :- सकळाही शक्ती क्रियास्वरुपें अधीना जालिया तरि परमेश्वर काइ करीति पां? ना माया कृपा व्यतिरिक्त सकळाही (शक्ती) दाना करीति : कृपाळु हितक्ष म्हणौनि : वांचौनि वंचकत्व बोलोंचि न ये : ॥
वचन :- माया कृपा व्यतिरिक्त परमेश्वरु तयासि सकळाही शक्ती दाना करीति ॥
टीका :- माया कृपा व्यतिरिक्त म्ह. या दोन्हीं वेगळिया करुनि : कां जें दिधलेयां दुसरा ईश्वर होईल : जीवातें उद्धरों लागैल : म्हणौनि अदया : यथा रावो सप्तांग राज्य दे परि उपभोग्य स्त्री नेदी : तथा या दोन्हीं पतिव्रता : जडांत कृपा थोर : चेतनांत माया थोर : दोहीचें कार्य अव्यक्तीं व्यक्तीं दिसे : ईश्वर अवतरविलें ते कृपाचि कीं :॥: सकळा म्ह. या दोन्हीं वांचौनि जडाचेतना :॥: सकळ म्ह. कळासहित ते सकळ : एकी चेतना : एकी सामथ्र्यरुपा : एक सामथ्र्य शाब्दीं आलें : एक अपरोक्षी : अनंत शक्ती सामथ्र्य हें आलें : तें दान तें सत्ता संपादुनीचि केलें : आपुली सत्ता सांडिली नाहीं : यथा पिता पुत्रा राज्य दे :॥: विशेषज्ञान देतखेवों शक्ती देति : यथा पत्रछेदनीन्यावो :॥: विशेषज्ञानियासि अवघिया शक्ती दिधलिया परि माया कृपा नाहीं : ज्ञान प्रेम न देववे : हे उणें : म्हणौनि ईश्वरीं संपूर्णा जीवीं अंशता : हेंचि एकासि दिधलेयां दुसरेया काइ : ना यें अर्थीं तिया अव्ययाही केलिया : ईश्वरीं आणि तयाचां ठाइं होणें उरणें आति : ॥५०॥

लापिका :- तरि दाना केलिया शक्ती तयातें कैसें मानीति पां? ना जैसा ईश्वर तैसाचि हा : अन्यथात्व एतुलेंहीं नाहीं : ॥
वचन :- मग तो सकळांसिही प्रमाणद्रव्य होए ॥
टीका :- प्रमाण द्रव्य म्ह. यथा ईश्वर : अवघेंयांही प्रमाण : तिया विशेषदृष्टीं यातें देखति परि स्वामीं हें ना : तें ईश्वरातेंचि : यथा रावो पुत्रा राज्य दे : मा परिवार जेणें भावें रायातें देखत होता तेणेंचि तयातेंही देखे : जैसा तो तैसाचि हा : परि तोचि न म्हणति : ॥५१॥

लापिका :- ऐसा आपणेयासमान करुनि मग परमेश्वर काइ करीति पां? ना अयोग्यता नाशार्थ चक्रवर्तीवत् उदास होति : मग तो कृपापात्र रंक होउनि किती एक काळ असे : ॥
वचन :- आद्यमळयुक्त चरमदेहस्थित कृपास्पदीभूत होउनि असे ॥
टीका :- चरम म्ह. एकी कडेंचें : विश्वेचे देह : सेवट केवळ कारणाचे जाणावें :॥: युक्त म्ह. मिसळला : कालवलेंपण : तादात्म्य तरि निवर्तलें परि वेगळा हों नेणें : यथा पशू : संसरणीं रिगावेया सामथ्र्य होते :॥: कृपा म्ह. योग्य : आणि आर्ती उत्कंठा प्राप्तीची :॥: कृपेसि योग्य जाला कृपा : आणिकें सुखें दुःखें (दुःख :) तो आर्तचि नव्हे :॥: चरमदेह तें योग्यताकारक होए कीं नव्हे : तो विचार : दोहीं अर्थीं उपपत्ती असति बहुता : परि एतुला आधार असे : प्राप्ती तें योग्यतादेहीं होआवी : आणि यें देहीं स्थित असे : ऐसा विशेष असे : या असणेयाचि तवं योग्यता बोलावी : आणि या देहाचे विश्वेसि दर्शन आति : एरी देवता नाहीं : कां तो प्रपंच नव्हे म्हणौनि :॥:॥: बाइचेनि देहें प्राप्ती करणें आति : असतिपरीचेया व्याख्यानातवं : भटां अपरोक्ष तियेचि देहीं यातवं : तियेचि देहीं तिन्हीं ज्ञानें देति : प्राप्ती करीति : चरमदेहस्थित ते पुरुषांही आणिक देह : तैसेचि स्त्रियेसि : ॥रामेश्वरबास॥ चरमदेह म्ह. तेचि मनुष्यदेह : ॥५२॥

लापिका :- ऐसेनि अव्हेरें अयोग्यता नासलियां मग परमेश्वर काइ करीति पां? ना प्राप्ती करावी ऐसी कृपा अंगीकरीति : ॥
वचन :- मग परमेश्वरु तयाचि कृपा करीति ॥
टीका :- कृपा करीति म्ह. अव्हेर उदासीनता होती तें जाउनि कणव उपजे : अंगीकार करीति : अनुकुळ होति : प्राप्ती करावी ऐसें उपजे : ॥५३॥

लापिका :- तरि कृपा स्वीकरौनि काइ करीति पां? ना तिये स्वशक्तीकरौनि अनादि संसारबीज अज्ञानरुप अविद्या छेदीति : ॥
वचन :- मग परमेश्वरु आपुलिया कृपाशक्तीकरौनि तयाची अनादि अविद्या छेदु करीति : मुळाविद्या छेदु करीति : अज्ञान छेदु करीति ॥
टीका :- मग अविद्या छेदणें हें नांतर्येक : शक्तीकरौनि म्ह. तियेतें अंगीकरुनि : मनोधर्म आणि दृष्टी : यथा प्रेम :॥: शक्ती आणि मनोधर्म :॥: मुळाविद्या म्ह. मुळीची म्हणौनि : एवं ऐसी ते तेचि होए : हे वचन प्रश्नानुरुप :॥: सकळां वोखटेयां गोमटेयां मूळ : इयेचें कार्य जीवत्व : सुख दुःख ग्राहकत्व : आणि अज्ञानाचें कार्य अन्यथाज्ञान :॥: अनादि अविद्या कां म्हणितलीं : जें आदिवंतीचेया अपेक्षा : तेचि कवण पां : ना आद्यमळ : तरि मुळाविद्या कां म्हणितली : जे अनादिची तरि होए परि दोन्ही वेगळाली : अविद्या आणि जीव ऐसें लाभैल : जैसे जीव आणि चैतन्य वेगळाली असति मग मिळति ऐसें होईल : म्हणौनि मुळाविद्या म्हणितलीं : दोहीं एकचि मूळ सांगितलें :॥: विशेषज्ञानियासि तीन अज्ञानें : भक्ताचें साधन : साध्य : अनुभूति :I।: अविद्येचें गोत्र : अनादि मूळ : अज्ञान : अन्यथाज्ञान : अन्यथाज्ञानीं जीवत्व : पांचपिसीं हे फिटे तैं तांदुळ आणि कोंडा :।I: ॥५४॥

लापिका :- तरि अन्यथाज्ञानादित्रय स्वतंत्र नासावें लागे आणि काइ? ना इयेचा नासीं तयाचा नाश : ॥
वचन :- अज्ञान छेदेकरौनि अन्यथाज्ञान त्रुटी होए : आद्यमळ त्रुटी होए : जीवत्व त्रुटी होए ॥
टीका :- अज्ञान छेदेंकरौनि म्ह. अज्ञान कारण, एर कार्य : अत्यंतिक नेणीव तें अज्ञान : मग आनातें आन म्हणें तें अन्यथाज्ञान : हें अज्ञानाचें कार्य : तयापासौनि उपजे म्हणौनि : त्रुटी म्ह. उपावो करावा न लगे : आपसयाचि कारणाचां नासीं नास : तें त्रुटी : तथा आद्यमळ त्रुटी : तथा जीवत्व त्रुटी : जीवत्व म्ह. जीवन क्रिये विषयो होइजे ऐसा अधिकार तयातें म्हणिजे : पदार्थाचां अज्ञानीं अन्यथाज्ञान : यथा दोराचां ठाइं सर्पबुद्धी : जीवत्व म्ह. यथा कृपे विषयो (विखो) होणें : अहंभावो उपजे ऐसी अधिकार भूमी : आत्मलब्धी तो अंकुर : तो अविद्येवांचौनि उपजेना :।I: अज्ञानी अन्यथाज्ञान तें एक अविद्या : एक स्वरुपींचें : तें अध्यासा कारण :॥: अविद्या तें मळासि कारण : अच्युताचें तें चतुर्विधा कारण : आपुलें तें अन्यथाज्ञाना कारण : दोनि नामें : उच्छेद तो प्रेमें : भक्ती नव्हे : ॥५५॥

लापिका :- हें बुझावेया गोसावीं स्वमुखेंचि न्याय दिधला : ॥
वचन :- तळिचील हाणितलेयां वरिचील पडे : उतरंडी गडबडी ॥
टीका :- मुदला धर्मियाचां नासी एराचां नाश : तो आधार आधेयत्वें : यथा उतरंडी गडबडी : बुडील आधार मोडलेयां वरील रचना मोडे : अविद्येवरी चौंची रचना : अज्ञान, अन्यथाज्ञान, आद्यमळ, जीवत्व : हे उपपत्ती बुझावेया श्रीमुखेंचि : ॥५६॥

लापिका :- एवं अविद्यादि दोष नासौनि परमेश्वर (काइ करीति पां? ना) चित्पारातें पाववीति : ॥
वचन :- या परी परमेश्वरु तयातें मायापारंगत करीति ॥
टीका :- पारंगत म्ह. तियेचिया स्वरुपांत असौनि वेगळा : यथा दुधांत चणे उदक घेति परि स्नेह नेघति : तैसा स्वरुपें वेगळा : तेथ असौनि परि जीव मायेसीं नातळे : ॥५७॥

लापिका :- मग परमेश्वर काइ करीति पां? ना प्राप्तीसमयीं ज्ञानादि सकळ साधन निवर्तौनि साध्य सुख (सौख्य) देति : ॥
वचन :- साध्याचिया प्राप्ती साधन निवर्ते ॥
टीका :- साध्याचिया प्राप्ती म्ह. यथा आनंद प्रकटति :॥: आनंद म्हणिजे : मागील कड पाहातां ईश्वरा एवढां हा नामिया : एरें दोन्हीं योग नामें : यथा आद्यमळ :॥:I।: साधन निवर्ते म्ह. सृष्ट्यंतरीची मुख्य चहु ज्ञानांतु जोडिली योग्यता आणि तें जेणें साधनें जोडिली तें ज्ञानशक्तीही स्वरुपातें पातलेयां परमेश्वर फेडीति या नावें निवर्ते : शाब्दींची घोडेवत् : कां तेथ आडळ बहुत : अपरोक्षींची दांडीवत् : सामान्यींची पाउवत् : विशेषींची तें वालिप स्थानी : एथ राणियेची उपपत्ती : राणी सेजार पातलेयां भोगसमयीं वालिप तें रायासीचि फेडणें लागे :।I:I।: तैसी परमेश्वर योग्यता फेडीति : गुण कां फेडावा : ना असिलेयां सुखासि आड राहे : यथा जवनिक : आणि हे जवं तवं केवळत्व नाहीं : म्हणौनि फेडणें लागे : दुसरे : साध्याचिया प्राप्ती साधन निवर्ते : हा साधनरुप लेप चरितार्थ होए एतुलेयासि वाचक :।I: ॥५८॥

लापिका :- तरि उभयासि सारिखीचि देति आणि काइ? ना भक्तासि असाधारण स्वसुखानुभूति रति देति : ज्ञानियाची व्यावृत्ती : ॥
वचन :- परमेश्वरु भक्तासि आपुली अनुभूति रति देति ॥
टीका :- भक्ता म्हणितलेया ज्ञानिया निराकरिला : आपुली अनुभूति म्ह. जे सुख आपण अनुभवीति तो अनुभव तयां देति : आपण स्वरुपीं रमति तेचि तया देति : म्हणौनि आपुली देति म्हणितले : दान दिधलें मा दिधलेंचि :॥: रति म्ह. रमण करितें द्वार : देति म्ह. अनुभवज्ञान : ॥५९॥

लापिका :- तरि इतर फळवत् एकदेशीत्वें देति आणि काइ? ना समग्र स्वरुप अवंचकत्वें सन्मुख देति : ॥
वचन :- भक्तासि आपणपे समोर देति ॥
टीका :- समोर म्ह. अभिमुख : यथा माता स्तनपान : दृष्टी आणि मनोधर्म आणि हातें स्पर्शणें (परिवसणे) :।: आपणपें म्ह. समग्र स्वरुप अवंचकत्वें : आंगे जीवें : जैसी स्त्री : हें अनुभवें जाणावें :I।: आपणपें देति मा जीव देत आहाति कीं : तें कार्याचि पुरतें : कार्य सरलेयां सवेंचि आणिकीकडे धांवताति :।: देवता देताति कीं : तें सप्रपंच दे : मागौता भोग सरलेयां घाली :।: परमेश्वरें दिधलें मा दिधलेंचि : काहीं न वंचीति : जीव सोच्य : अप्रकाशवंत : कष्मळ : दोषवंत : ऐसेयासि रमण देताति तरि काइ परमेश्वरही रमति : म्हणें ना : तो स्वस्वरुपीं रमे :।I: ॥६०॥

लापिका :- भक्तासि ऐसें दिधलें तरि ज्ञानियासि कैसें देति पां? ना आनंदमय ब्रह्मस्वरुपेंसीं समरस (सामरस) करीति : ॥
वचन :- ज्ञानियासि ब्रह्मींभूत करीति ॥
टीका :- भूत म्ह. मर्दनें, मिसळणे : यथा तेलातुपा ऐक्यता : ॥६१॥

लापिका :- एवं ब्रह्ममज्जनत्वें अप्राप्ती आदि सकळ दुःखावेगळा आत्यंतिक करीति : यथा शरणागत : ॥
वचन :- ज्ञानियासि पाठीसि घालीति ॥
टीका :- पाठींसी म्ह. आत्यंतिक दुःखापासौनि रक्षिला : यथा शरणागत : ॥६२॥

लापिका :- दुःख तरि निवर्तलें परि अनुभव-अज्ञान असेतवं सुख न फवे हेंही संसार-बंधनचि : जे विशेषज्ञानें न नसेचि ; अविद्येचिये पांति न फिटेचि ; ब्रह्मैक्यता न वचेचि : तें ईश्वरें अनुभवज्ञाने नासिलें : नित्यमुक्त करुनि आनंदमय ब्रह्मपदी स्थापिलें : ॥
वचन :- ज्ञानियासि मोक्ष देति ॥
टीका :- मोक्ष म्ह. बंधत्रुटी आणि ब्रह्मस्वरुपीं मिसळणें : देति म्ह. नित्यत्वें : संसारापासौनि सोडविला परि यदा कदा काळेंकरौनि च्युति होईल, बंधन लागैल आणि काइ : तरि तें देति एणें निराकरिलें : सदासर्वदा मोकळा केला : कां ईश्वरीचें नित्य एणें न्यायें : ॥६३॥

लापिका :- तरि चैतन्यफळवत् एकदेशीं मज्जन करीति आणि काइ? ना कृपाळुपणें समग्रे स्वरुपीं मिसळौनि अगाधत्वें हारपौनि गेला : ॥
वचन :- ज्ञानिया ब्रह्मीं मिसळे : बुडी दे ॥
टीका :- मिसळे म्ह. समग्र स्वरुपांत : मागिल भूत शब्द एकदेशीं जाला :।: बुडणें म्ह. हारपलेपण : अगाधत्वें : एऱ्हवीं समग्र स्वरुपीं मिसळला तरि व्यापक ब्रह्मत्व आपजैल तें निराकरिलें : यथा तळेयामध्यें तेलाचा थेंब घातला तो हारपौनि गेला उदकाचेनि बाहुल्यें : ॥६४॥

लापिका :- एवं ज्ञानियाचा अभेदानुभव सांगितला तरि भक्ताचा कैसा पां? ना तो यथातथ्ये निरुपिता जीव सहसा बुझेना म्हणौनि भेदाभेद निराकरिले : तो अप्रतिरुप : सद्गुरुप्रसादें जाणिजे : ॥
वचन :- भक्त भेदें अनुभवी की अभेदें अनुभवी ऐसा विकल्प तेथ करों न ये ॥
टीका :- भेद म्ह. वेगळेपण : अभेद म्ह. मिसळौनि : विकल्प करुं न येति म्ह. तो अनुभव कैसा हें जीव सहसा बुझेना : म्हणौनि ऐसें किंवा ऐसें ऐसिया कल्पना न कराविया : कां जें मिसळौनि उरीं हें विरुद्ध बोलता : परि मिसळौनि हेंचि होए : कां जें ज्ञानियासि मिळणी आणि यासि वेगळेपण हें दूषण : आणि भलतिये प्राप्ती मिळणीविन सुख नाहीं :॥:॥: भक्ताचिये अनुभूतीसि एराचें भूतत्व : रतिसी मोक्ष : समोरत्वें पाठीं : आपणपे मिसळे : बुडियेसि भेदाभेद : तो विरुद्धाधर्मा सारिखा : तेथ मायाचि विरुद्ध : वांचौनि तो सर्वदा सन्मुख : तो जरि विमुख जाला तरि जीव मेले कीं : एका आपणपें दिधलें परि आणिकालागौनि अवेच : यथा दिवेयापासौनि अनंत नेइजति : ॥६५॥

लापिका :- या परी कृपासागर परमेश्वरु श्रीचक्रधरु ज्ञानिया भक्तासि उध्र्व करुनि कैवल्यपद प्राप्ती करीति : ॥
वचन :- या परी परमेश्वरु श्रीचक्रधरु जीवांते उद्धरीति ॥
टीका :- या परी म्ह. मागां उपक्रमोपसंहार सांगितला : जरि परमेश्वर तरि क्रमेंचि : दुसरें : मति जालेयां एसणें गोमटें देति : आपणपें : म्हणौनिचि परमेश्वर : जीवातें म्हणितलेया देवतां व्यावृत्ती : श्रीचक्रधर म्ह. भटांचें बोलणें : ऐसें करिता कोण तें नाम सांगितलें :॥: उद्धरी म्ह. उध्र्व करी : ज्ञानिया भक्तातें : ॥६६॥
